Nom : technicalCriteria
Description : Critères techniques.
Choix : Non
  • technicalCriteria
    • limitCriteria : limitCriteria

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : limitCriteria
Description : Nombre maximum d'enregistrements à retourner.
Commentaire :

Ce critère permet de limiter le nombre d’enregistrements renvoyés par un service.

La limite d’enregistrements renvoyés prend en compte la valeur minimale entre ce critère et le paramétrage mis en place pour la totalité des services via le Montant 1 du paramètre BS_RESPONSE en nomenclature 098.

Nom : limitCriteria
Description : Nombre maximum d'enregistrements à retourner.
Format : integer
