  • ceilingUsed
Nom : ceilingUsed
Description : Ce type définit le détail du plafond utilisé : plafond limitant entre limite client, limite groupe ou autorisation plafonnée du client,possédant le plus petit montant disponible en devise nationale.
Choix : Non
  • ceilingUsed
    • ceilingNumber : limitNumber
    • ceilingType : ceilingType
    • dueDate : date
    • referenceCurrency : currencyCode
    • ceilingAmount : decimal19_4
    • ceilingAmountUsed : decimal19_4
    • mergerBalances : boolean

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
