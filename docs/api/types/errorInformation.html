  • errorInformation
Nom : errorInformation
Description : Informations sur l'erreur rencontrée, ne sont présentes que pour les messages d'erreur (nature = "ERROR").
Choix : Non
  • errorInformation
    • errorCode : string
    • convertedCode : string
    • errorMessage : string

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : errorCode
Description : Code original de l'erreur, provenant du SGBD (Informix, Oracle) ou de Genero.
Commentaire :

Ce code correspond au code de l'erreur technique retournée par le test de connexion.

Nom : string
Nom : convertedCode
Description : Code de l'erreur converti pour homogénéiser les codes erreurs entre les SGBD Informix et Oracle.
Nom : string
Nom : errorMessage
Description : Message de l'erreur originale provenant du SGBD ou de Genero.
Nom : string
