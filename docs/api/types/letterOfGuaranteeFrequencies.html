  • letterOfGuaranteeFrequencies
Nom : letterOfGuaranteeFrequencies
Description : Ce type contient les périodicités d'un dossier de caution.
Choix : Non
  • letterOfGuaranteeFrequencies
    • frequencyOfCommission : letterOfGuaranteeFrequency
    • nextFrequencyOfCommission : letterOfGuaranteeNextFrequency
    • frequencyOfDeductions : letterOfGuaranteeFrequencyOfDeductions

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : frequencyOfCommission
Description : Périodicité
Nom : letterOfGuaranteeFrequency
Description : Périodicité des cautions.
Format : string
Valeurs possibles :
H 8 mois
I 5 mois
A Annuelle
B Bimestrielle
C Mise en place
D 10 mois
E 7 mois
O 11 mois
P 9 mois
Q 4 mois
R Mise en place + échéance
S Semestrielle
T Trimestrielle
M Mensuelle
Nom : nextFrequencyOfCommission
Description : Périodicité suivante
Nom : letterOfGuaranteeNextFrequency
Description : périodicité suivante des cautions.
Format : string
Valeurs possibles :
A Anuelle
S Semestrielle
T Trimestrielle
M Mensuelle
Nom : frequencyOfDeductions
Description : Périodicité des retenues de garantie
Nom : letterOfGuaranteeFrequencyOfDeductions
Description : Périodicité des retenues de garantie des cautions.
Format : string
Valeurs possibles :
M Mensuelle
A Annuelle
S Semestrielle
T Trimestrielle
