  • dosprtCreateDepositAccount
Nom : dosprtCreateDepositAccount
Description : Garantie de compte de dépôt pour le calcul de l'EIR.
Choix : Non
  • dosprtCreateDepositAccount
    • [C] collateralAmountOrRate : collateralAmountOrRate
      • collateralAmount : decimal19_4
      • collateralRate : decimal10_7
    • taxable : boolean
    • depositAccountPeriodicity : depositAccountPeriodicity
    • creditInterestRate : decimal10_7
    • taxRate : decimal10_7
    • rateOfDeductionAtSource : decimal10_7
    • interestCapitalisation : char1

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : collateralAmountOrRate
Description : Montant ou taux de la garantie.
Commentaire :

Si ni le montant de la garantie, ni le taux de la garantie ne sont renseignés, alors le montant de la garantie sera affecté par défaut avec la valeur paramétrée pour la condition MONGAR.

Nom : collateralAmountOrRate
Description : Montant ou taux de garantie.
Nom : collateralAmount
Description : Montant de garantie.
Commentaire :

S’il est renseigné, le montant de la garantie doit être compris dans les bornes minimum et maximum paramétrées pour la condition MONGAR.

Nom : decimal19_4
Description : Type décimal à 19 chiffres dont 4 décimales.
Format : decimal
Nombre total de chiffres : 19
Nom : collateralRate
Description : Taux de garantie.
Commentaire :

Si le taux de la garantie est renseigné, alors le montant de la garantie est calculé à partir du montant du prêt multiplié par le taux de garantie divisé par 100. Le montant de la garantie calculé doit être compris dans les bornes minimum et maximum paramétrées pour la condition MONGAR.

Nom : decimal10_7
Description : Type décimal à 10 chiffres dont 7 décimales.
Format : decimal
Nombre total de chiffres : 10
Nom : taxable
Description : Taxable.
Commentaire :

Si non renseigné, le caractère taxable est affecté par défaut à FALSE.

Nom : boolean
Nom : depositAccountPeriodicity
Description : Périodicité du compte de dépôt.
Commentaire :

Si la périodicité du compte de dépôt n’est pas renseignée alors la périodicité est affectée par défaut à mensuelle (M).

Si la périodicitérenseignée est égale à Epargne (E) alors la périodicité est forcée à Annuelle (A).

Nom : depositAccountPeriodicity
Description : Périodicité du compte de dépôt.
Format : string
Valeurs possibles :
M Mensuelle
S Semestrielle
T Trimestrielle
E Epargne
A Annuelle
Nom : creditInterestRate
Description : Taux d'intérêts créditeur.
Commentaire :

Si le taux d'intérêts créditeur n’est pas renseigné, il sera affecté par défaut avec la valeur paramétrée pour la condition CREGAR.

Nom : decimal10_7
Description : Type décimal à 10 chiffres dont 7 décimales.
Format : decimal
Nombre total de chiffres : 10
Nom : taxRate
Description : Taux de taxe.
Commentaire :

Si le taux de taxe n’est pas renseigné, il sera affecté par défaut avec la valeur paramétrée pour la condition TAXGAR.

S’il est renseigné, Le taux de taxe doit être compris dans les bornes minimum et maximum paramétrées pour la condition TAXGAR.

Nom : decimal10_7
Description : Type décimal à 10 chiffres dont 7 décimales.
Format : decimal
Nombre total de chiffres : 10
Nom : rateOfDeductionAtSource
Description : Taux de prélèvement libératoire.
Commentaire :

Si le taux de prélèvement libératoire n’est pas renseigné, il sera affecté par défaut avec la valeur paramétrée pour la condition PLIGAR.

S’il est renseigné, le taux de prélèvement libératoire doit être compris dans les bornes minimum et maximum paramétrées pour la condition PLIGAR.

Nom : decimal10_7
Description : Type décimal à 10 chiffres dont 7 décimales.
Format : decimal
Nombre total de chiffres : 10
Nom : interestCapitalisation
Description : Capitalisation des intérêts.
Commentaire :

S’il n’est pas renseigné, l’indicateur de capitalisation est affecté à la valeur N.

Nom : char1
Description : Type avec un nombre de caractères égal à 1.
Format : string
Longueur fixe : 1
