Nom : otherAccount
Description : Règlement par un autre compte.
Commentaire :

La devise de cet autre compte doit être égale à la devise du dossier de crédit.

Le compte doit exister dans la table des comptes et ne doit pas être fermé.

Le compte doit appartenir au fournisseur (bénéficiaire).

Le nombre de caractères du numéro de compte doit être égal  au nombre paramétré dans le Montant 1 du paramètre COMPTE en nomenclature 999.

Nom : costAmount
Description : Montant hors taxe (HT).
Commentaire :

Si le montant n’est pas renseigné, il sera affecté  avec la valeur par défaut paramétrée dans la condition CM1ttt ou M1PRT.

Le montant doit être compris entre les valeurs minimum et maximum paramétrées dans la condition CM1ttt ou M1PRT.

Le terme ttt ci-dessus correspond au code du type de prêt.

Le montant TTC est alors calculé en ajoutant le montant de la taxe au montant HT.
