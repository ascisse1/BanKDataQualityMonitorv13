Nom : dosprtCreateLoanFileCollateralForEIRCalculation
Description : Garantie pour le calcul de l'EIR.
Choix : Oui
  • dosprtCreateLoanFileCollateralForEIRCalculation

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Description : Montant ou taux de la garantie.

Nom : collateralRate
Description : Taux de garantie.
Commentaire :

Si le taux de la garantie est renseigné, alors le montant de la garantie est calculé à partir du montant du prêt multiplié par le taux de garantie divisé par 100. Le montant de la garantie calculé doit être compris dans les bornes minimum et maximum paramétrées pour la condition MONGAR.
