Nom : branch
Description : Agence de saisie du virement.
Commentaire :

Cas particulier des virements SIOP : la notion d'agence de saisie n'existant pas pour les virements SIOP, le service filtre sur l'agence du compte du donneur d’ordre. Ainsi, si les 2 informations "Agence de saisie" et "Agence du compte du donneur d’ordre" sont renseignées dans la requête, alors elles doivent être identiques (sinon le service ne restitue aucun virement SIOP).
