Nom : customerLegalInformationCreate
Description : Informations légales d'une société.

Pour un client de type Société (2 ou 3).

Choix : Non
  • customerLegalInformationCreate
    • tradeRegisterNumber : charMax20
    • deliveryDateOfTradeRegister : date
    • tradeRegisterDeliveryPlace : charMax30
    • validityDateOfTradeRegister : date
    • chamberOfCommerceCode : chamberOfCommerceCode
    • licenseNumber : charMax20
    • validityDateOfLicense : date

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : tradeRegisterNumber
Description : Numéro de registre de commerce.
Commentaire :

Ce numéro correspond soit au numéro de registre de commerce, soit au numéro SIREN si le paramètre BANQ_DE_FR existe en nomenclature 098.

Si le contrôle d'unicité du numéro de registre de commerce est demandé dans le Libellé 2 (égal à O (contrôle non bloquant) ou B (contrôle bloquant uniquement si l’utilisateur n’a pas le niveau de forçage requis indiqué dans le Montant 1)) du paramètre CTRIDENCLI en nomenclature 098, alors le numéro de registre de commerce ne doit pas déjà exister dans la table des clients.

Valeur par défaut : nrc

Nom : charMax20
Description : Chaîne composée d'au plus 20 caractères.
Format : string
Longueur mini-maxi : 1 - 20
Nom : deliveryDateOfTradeRegister
Description : Date de délivrance du registre de commerce.
Commentaire :

La date de délivrance du registre de commerce ne peut être renseignée que si le numéro de registre de commerce l’est.

La date de délivrance du registre de commerce doit être postérieure ou égale à la date comptable.

Valeur par défaut (contrôle de l’obligation de saisie uniquement) : drc

Nom : date

Description : Lieu de délivrance du registre de commerce.
Commentaire :

Le lieu de délivrance du registre de commerce ne peut être renseigné que si le numéro de registre de commerce l’est.

Valeur par défaut : lrc

Nom : charMax30
Description : Chaîne composée d'au plus 30 caractères.
Format : string
Longueur mini-maxi : 1 - 30
Nom : validityDateOfTradeRegister
Description : Date de validité du registre de commerce.
Commentaire :

La date de validité du registre de commerce ne peut être renseignée que si le numéro de registre de commerce l’est.

La date de validité du registre de commerce doit être postérieure ou égale à la date comptable.

Si la date de validité du registre de commerce est obligatoire (Montant 1 différent de 1 dans le paramètre CTRLDIVCLI de la nomenclature 098) alors l’année de la date de validité du registre de commerce ne doit excéder ni l’année de la date comptable + 200, ni l’année de la date de délivrance du registre de commerce + 200.

Valeur par défaut (contrôle de l’obligation de saisie uniquement) : vrc

Nom : date

Format : string
Longueur mini-maxi : 1 - 20
Nom : validityDateOfLicense
Description : Date de validité de la patente.
Commentaire :

La date de validité de la patente ne peut être renseignée que si le numéro de patente l’est.

La date de validité de la patente doit être postérieure ou égale à la date comptable et être valide : l’année de la date de validité de la patente ne doit pas être supérieure à l’année de la date comptable + 200.

Valeur par défaut (contrôle de l’obligation de saisie uniquement) : vpa
