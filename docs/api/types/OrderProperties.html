  • OrderProperties
Nom : OrderProperties
Description : Informations sur les modifications d'un ordre de paiement différé/permanent.
Choix : Non
  • OrderProperties
    • Tp : TypeCode
    • Md : ModeCode
    • ChngInf : ChangeInformation1
      • ChngId : Max35Text
      • [C] OrgnlOrdrRef : OriginalOrderReference1
        • AmplLotId : Max38Text
        • SnglOrdrRef : Max38Text
      • RsnInf : ReasonInformation
        • Rsn : Max35Text
        • AddtlInf : Max140Text
    • StdgOrdrInf : StandingOrderInformation1
      • Frqcy : FrequencyCode
      • ExctnDay : Max2NumericText
      • NbOfInstlments : Max3Text
      • FrstInstlmentDt : ISODate
      • LstInstlmentDt : ISODate
      • Suspn : Suspension
        • From : ISODate
        • To : ISODate

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
