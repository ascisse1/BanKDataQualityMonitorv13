Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : orderNumber
Description : Numéro d'ordre du compte supplémentaire.
Commentaire :

Pour une carte, il peut y avoir plusieurs comptes supplémentaires : le numéro d’ordre sert à les différencier.

Il doit donc être unique pour chaque compte. Egalement, les numéros d'ordre doivent être utilisés dans l’ordre croissant (il est impossible d’avoir un compte supplémentaire avec un numéro d’ordre à 2 si le 1 n’existe pas).

Enfin, un nombre de comptes supplémentaires maximum est paramétré au niveau du type de carte.
