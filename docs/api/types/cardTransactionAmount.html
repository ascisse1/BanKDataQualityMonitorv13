  • cardTransactionAmount
Nom : cardTransactionAmount
Description : Montant de la transaction, exprimé dans plusieurs devises.

La devise des montants doit exister dans les nomenclatures 002 (devises alphanumériques) et 005 (devises numériques).

Si la désignation d’une devise n’est pas considérée pour la sélection de cette devise.

Le nombre de décimales fourni dans le flux n’est pas considéré, c’est le Montant 1 issu du paramétrage de la devise qui est pris en compte.

Au moins un montant doit être renseigné.

Choix : Non
  • cardTransactionAmount
    • expressionCurrencyType : cardTransactionAmountExpressionCurrencyType
    • amount : amount
      • amount : decimal19_4
      • currency : amountCurrency
        • currency : simpleCurrency
          • alphaCode : char3
          • numericCode : char3
          • designation : charMax30
        • numberOfDecimals : integerBetween0And4

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
