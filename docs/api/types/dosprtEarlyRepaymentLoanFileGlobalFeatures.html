Nom : dosprtEarlyRepaymentLoanFileGlobalFeatures
Description : Caractéristiques générales d'une tombée anticipée de dossier de prêt

      • paymentCommission : decimal19_4

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.
