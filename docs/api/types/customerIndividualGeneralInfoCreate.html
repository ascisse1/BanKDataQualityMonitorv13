Nom : firstname
Description : Prénom.
Commentaire :

Si le contrôle spécifique du prénom est demandé dans le Montant 1 (égal à 1) du paramètre CTRLNOMCLI en nomenclature 098, alors le prénom ne doit contenir que des caractères alphabétiques.

Le prénom est modifiable si le niveau de forçage de l’utilisateur est supérieur ou égal au niveau de forçage paramétré dans le Montant 1 du paramètre MODNOM en nomenclature 099.

Lors de la modification d’un client, si le prénom est modifié, alors il est mis à jour pour les :

  • dirigeants correspondant au client dans la table des dirigeants,
  • actionnaires correspondant au client dans la table des actionnaires,
  • comptes joints du client dans la table des comptes joints.

Valeur par défaut : pre

Nom : holderLegalCapacity
Description : Capacité juridique du client.
Commentaire :

Si la date de naissance est renseignée et que l’âge de la majorité est paramétré dans le Montant 1 du paramètre CTRLMAJCLI de la nomenclature 098 alors la capacité juridique du client doit correspondre à la capacité juridique d’un mineur (paramétrée dans le Libellé 2), ou à la capacité juridique d’un majeur (paramétrée dans le Libellé 3)

Les codes de capacité juridique d’un mineur ou d’un majeur doivent exister en nomenclature 172.

Valeur par défaut : capj

Nom : legalCapacityCode
Description : Code de capacité juridique.
Format : string
Longueur fixe : 1
Nom : applicationDateOfLegalCapacity
Description : Date d'application de la capacité juridique.
Commentaire :

La date d’application de la capacité juridique ne peut être renseignée que si la capacité juridique l’est.

Si elles sont renseignées, la date d’application de la capacité juridique doit être postérieure à la date de naissance du client.

Elle doit être une date valide : l’année de la date d’application de la capacité juridique ne doit pas être supérieure à l’année de la date comptable + 200.

Si la date d’application de la capacité juridique est postérieure à la date comptable, les jours fériés de l’année correspondante doivent être paramétrés dans la table des jours fériés.

Valeur par défaut (contrôle de l’obligation de saisie uniquement) : dcapj
