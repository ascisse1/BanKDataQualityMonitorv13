Nom : levellingAmountType
Description : Type de montant de nivellement.
Format : string
Valeurs possibles :

C Montant total du crédit
S Solde
T Solde minoré
U Pourcentage du solde
F Montant fixe
P Pourcentage du montant du crédit

Liste des valeurs possibles :

  • ¿ M = Montant du crédit minoré
  • ¿ F = Montant fixe
  • ¿ P = Pourcentage du montant du crédit
