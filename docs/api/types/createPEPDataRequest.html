  • createPEPDataRequest
Nom : createPEPDataRequest
Description : Requête pour la création des informations PPE d'un client ou d'un tiers.
Choix : Non
  • createPEPDataRequest
    • customer : customerCode
    • PEPCustomerList : PEPCustomerList
      • [M] PEPcustomerDetails : PEPCustomerDetails
        • PEPCustomerType : customerOrThirdPartyType
        • attachedCustomerCode : customerCode
        • LinkCode : char3
        • PEPPosition : char3
        • RiskLevel : decimal1_0
        • mandateBeginDate : date
        • mandateEndDate : date
        • ActivePEP : decimal1_0
        • DeclarationList : DeclarationList
          • [M] DeclarationDetails : DeclarationDetails
            • DeclarationDate : date
            • DeclarationNature : char1
            • DeclarationComments : charMax250
    • comment : charMax250

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
