Nom : limitCriteria
Description : Nombre maximum d'enregistrements à retourner.
Format : integer

Ce critère permet de limiter le nombre d’enregistrements renvoyés par un service.
La limite d’enregistrements renvoyés prend en compte la valeur minimale entre ce critère et le paramétrage mis en place pour la totalité des services via le Montant 1 du paramètre BS_RESPONSE en nomenclature 098.
