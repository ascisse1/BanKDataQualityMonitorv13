Nom : repaymentAccount
Description : Compte de remboursement dans la banque.
Commentaire :

Le compte de remboursement ne peut appartenir à une agence de la banque autre  que celle du dossier de crédit, sauf si le Taux 2 du paramètre CDTPAR en nomenclature 098 l’autorise (valeur à 1).

La devise du compte de remboursement doit être égale à la devise de remboursement paramétrée pour le type de prêt et la nature de compte 005.

Le nombre de caractères du numéro de compte doit être égal  au nombre paramétré dans le Montant 1 du paramètre COMPTE en nomenclature 999.

Le suffixe du compte doit être renseigné si le  Libellé 5 est égal à 1 pour le paramètre COMPTE en nomenclature 999.

Le compte de remboursement doit exister dans la table des comptes. Le client détenteur du compte doit correspondre au client de remboursement.

Le compte de remboursement ne doit être ni fermé ni en instance de fermeture si le forçage d’utilisation d’un compte fermé ou en instance de fermeture n’est pas autorisé.

Le chapitre comptable du compte de remboursement doit être autorisé, dans la balise suivante.
