  • accountAddressFormat
Nom : accountAddressFormat
Description : Format de l'adresse d'un compte.
Format : string
Valeurs possibles :
CX Cedex
SE Service
GU Guichet
GE Géographique
EM Email
TR Transport
CA Casier
SP Secteur postal
BP Boite postale

Liste des valeurs possibles :

  • BP = Boîte postale
  • SP = Secteur postal
  • GU = Guichet
  • CA = Casier
  • SE = Service
  • TR = Transport
  • EM = Email
  • GE = Géographique
  • CX = Cedex

Le format de l’adresse doit exister en nomenclature 181 et être géré par la banque (Montant 1 à 1).
