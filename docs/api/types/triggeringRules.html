Nom : triggeringRules
Description : Règles de déclenchement d¿un nivellement ou d¿un transfert de compte à compte.
Choix : Non
  • triggeringRules
    • triggerType : triggerType
    • specificTriggerLoans : triggeringRulesOfTransferRateOnLoans
      • allLoans : boolean
      • amountType : levellingAmountType
    • triggerThreshold : decimal19_4

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : triggerType
Nom : triggerType
Description : Type de déclenchement de nivellement.
Format : string
Valeurs possibles :
C Sur les crédits
U Sur un pourcentage du solde
A Sur solde avec autorisation de découvert
S Sur le solde
Nom : specificTriggerLoans
Description : Règles de déclenchement d¿un transfert au débit sur les crédits.
Commentaire :

Cette balise n’est présente que si le type de déclenchement est égal à C (sur les crédits).
