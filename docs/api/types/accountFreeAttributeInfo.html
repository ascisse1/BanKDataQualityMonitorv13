Nom : accountFreeAttributeInfo
Description : Information sur l'attribut libre.

Les attributs libres d’un compte et toutes ses informations sont définis dans la nomenclature 168.

Choix : Non
  • accountFreeAttributeInfo
    • code : charMax10
    • designation : charMax30
    • type : accountFreeAttributeType
    • dataFormat : freeAttributeFormat
    • mandatoryField : boolean
    • managementMethod : additionalDataManagementMethod
    • maxLength : short
    • numberOfDecimals : short
    • separator : boolean
    • existenceControl : short
    • displayOrder : short

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
