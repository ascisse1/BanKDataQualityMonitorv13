Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : behaviourIfCardStoppage
Description : Comportement en cas d'opposition sur la carte.
Commentaire :

Par défaut, les oppositions sur la carte ne sont pas contrôlées.

Si cette balise est renseignée, alors les contrôles sont activés et le comportement spécifié sera adopté.
