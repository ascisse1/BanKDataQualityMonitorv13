Nom : additionalDescriptionList
Description : Liste des libellés complémentaires.
Choix : Non
  • additionalDescriptionList
    • [M] additionalDescription : additionalDescription
      • orderNumber : decimal2_0
      • description : charMax65

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
