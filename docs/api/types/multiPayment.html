Nom : multiPaymentInvoices
Description : Liste des factures à régler pour le multi-paiement.
Commentaire :

La liste des factures n’est pas obligatoire pour un multi-paiement.

Si elle n’est pas renseignée, alors le fournisseur se chargera de répartir le montant payé sur les différentes factures du client.
