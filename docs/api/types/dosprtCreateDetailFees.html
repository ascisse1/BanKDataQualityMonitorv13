Nom : dosprtCreateDetailFees
Description : Frais.
Choix : Non
  • dosprtCreateDetailFees
    • [C] feesAmountOrRate : amountOrRate
      • amount : decimal19_4
      • rate : decimal8_6
    • feesMaturity : dosprtDetailMaturityType
    • feesPeriodicity : dosprtDetailPeriodicityType
    • feesCollectionMethod : dosprtDetailCollectionMethodType

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : feesAmountOrRate
Description : Montant ou taux de frais.
Commentaire :

La balise renseignée (Taux ou Montant) doit être en accord avec le paramétrage (M ou T) des frais dans les conditions FRAttt ou FRAPRT où ttt correspond au code du type de prêt.

Le montant ou le taux des frais doit être compris entre la valeur minimum et la valeur maximum de ces conditions.

Nom : amountOrRate
Description : Montant ou taux.

Description : Montant.
Nom : decimal19_4
Description : Type décimal à 19 chiffres dont 4 décimales.
Format : decimal
Nombre total de chiffres : 19
Nom : rate
Description : Taux.
Nom : decimal8_6
Description : Type décimal à 8 chiffres dont 6 décimales.
Format : decimal
Nombre total de chiffres : 8

Nom : feesPeriodicity
Description : Périodicité des frais.
Commentaire :

Valeur par défaut :

  • per_fra

La périodicité des  frais, si elle est renseignée, doit être un multiple du nombre de mois du prêt (durée du prêt calculée par différence entre la date de dernière et la date de première échéance, ou durée du prêt renseignée directement par le nombre d'échéances).

Nom : dosprtDetailPeriodicityType
Description : Périodicité.
Format : integer
Min-max inclusive : 1 - 12
