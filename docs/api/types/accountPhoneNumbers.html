Nom : accountIdentifier
Description : Identifiant du compte.
Commentaire :

Le compte n’est pas obligatoire si le service est appelé par les services createAccount et modifyAccount, et est obligatoire dans tous les autres cas.

S’il est renseigné, le compte doit exister dans la table des comptes et doit être un compte de clientèle. Le client référencé pour ce compte doit exister dans la table des clients.

Nom : phoneType
Description : Type de téléphone.
Commentaire :

Le type de téléphone du compte doit exister en nomenclature 182. Si une valeur est renseignée dans le Montant 3, elle doit correspondre au type de client du compte : 1 pour un client de type Particulier (1), 2 pour un client de type Société (2) ou Entreprise individuelle (3).

Un seul numéro de téléphone lié au compte est possible par type de téléphone.

Nom : charMax3

Format : string
Longueur mini-maxi : 1 - 3
Nom : phoneNumber
Description : Numéro de téléphone.
Commentaire :

Si une valeur est renseignée dans le Taux 4 en nomenclature 182 pour le type de téléphone, le masque de saisie du numéro de téléphone doit être le suivant :

  • Le premier caractère peut être un entier ou un "+".
  • Les autres caractères peuvent être un entier, un ".", un "/" ou un "-".

Si une valeur est renseignée dans le Taux 5 en nomenclature 182 pour le type de téléphone, la longueur du numéro de téléphone doit être inférieure ou égale à cette valeur.
