Nom : firstYearRate
Description : Taux de perception des commissions pour la première année.
Commentaire :

Par défaut, le taux de perception des commissions pour la première année est affecté avec le taux de  la condition CG1PRT pour le code opération enregistré au niveau de la nature d'évènement dans le Montant 1 du paramètre COGEA1 en nomenclature 052.

Les bornes minimale et maximale de la condition doivent être respectées.

Si la commission est taxable et que le compte ou le client de remboursement sont taxables, alors le taux de cette taxe est affecté avec le taux de  la condition TTXCG1.

Nom : nextPeriodRate
Description : Taux de perception des commissions sur les années suivantes.
Commentaire :

Par défaut, le taux de perception des commissions pour la première année est affecté avec le taux de  la condition CG2PRT pour le code opération enregistré au niveau de la nature d'évènement dans le Montant 1 du paramètre COGEA2 en nomenclature 052.

Les bornes minimale et maximale de la condition doivent être respectées.

Si cette  commission est taxable et que le compte ou le client de remboursement sont taxables, alors le taux de cette taxe est affecté avec le taux de  la condition TTXCG2.
