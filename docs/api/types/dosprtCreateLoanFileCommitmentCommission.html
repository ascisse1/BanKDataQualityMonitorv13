Nom : dosprtCreateLoanFileCommitmentCommission
Description : Commission d'engagement.
Choix : Non
  • dosprtCreateLoanFileCommitmentCommission
    • calculationDate : date
    • commissionRate : decimal8_6

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : calculationDate
Description : Date de début de calcul.
Commentaire :

La date doit être supérieure ou égale à la date de mise en place du dossier.

La date doit être inférieure ou égale à la date de dernière échéance du dossier.

Nom : date
Nom : commissionRate
Description : Taux de la commission d'engagement.
Commentaire :

Par défaut, le taux de la commission d’engagement est affectée avec le taux de  la condition MOEPRT pour le code opération enregistré au niveau du type de prêt.

Les bornes minimale et maximale de la condition doivent être respectées.

Si la commission est taxable et que le client de remboursement est  taxable, alors le taux de cette taxe est affecté avec le taux de  la condition TTAXEN.

Nom : decimal8_6
Description : Type décimal à 8 chiffres dont 6 décimales.
Format : decimal
Nombre total de chiffres : 8
