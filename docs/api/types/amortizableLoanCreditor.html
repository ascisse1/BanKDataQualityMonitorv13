Nom : creditorIdentification
Description : Identification du créancier.
Commentaire :

A l'intérieur de ce type, la saisie d'un code BIC implique la saisie d'un code IBAN et inversement.

Ces informations ne doivent être renseignées que si le paramètre CDTSANSBQE est positionné en nomenclature 098.
