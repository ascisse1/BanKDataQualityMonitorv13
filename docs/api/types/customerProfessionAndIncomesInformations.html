Nom : hireDate
Description : Date d'embauche.
Commentaire :

Si un nombre de jours est paramétré dans le Montant 1 de la nomenclature 098 pour le paramètre CONTRDEMB, alors la date d’embauche doit être inférieure ou égale à la date comptable à laquelle on ajoute ce nombre de jours.

Si un nombre de jours est paramétré dans le Montant 2 de la nomenclature 098 pour le paramètre CONTRDEMB et si le client est un client collectif (Montant 2 renseigné à 2 dans la nomenclature 036 pour le code intitulé du client), alors la date d’embauche doit être supérieure ou égale à la date comptable à laquelle on retire ce nombre de jours.

La date d’embauche doit être supérieure à la date de naissance du client.
