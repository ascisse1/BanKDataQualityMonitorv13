Nom : letterOfCreditShipmentAndDelivery
Description : Détail des informations d'expédition et de livraison des dossiers de lettre de crédit.
Choix : Non

Légende :

  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.
