  • complementaryCardData
Nom : complementaryCardData
Description : Données complémentaires de la carte.
Choix : Non
  • complementaryCardData
    • cardComplementaryDataNature : charMax10
    • cardComplementaryDataValue : charMax255

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : cardComplementaryDataNature
Description : Nature de la donnée complémentaire.
Commentaire :

La nature de donnée complémentaire doit être définie en nomenclature 378.
Elle doit également être définie au niveau du type de carte.
 

Nom : charMax10
Description : Chaîne composée d'au plus 10 caractères.
Format : string
Longueur mini-maxi : 1 - 10
Nom : cardComplementaryDataValue
Description : Valeur de la donnée complémentaire.
Commentaire :

La valeur doit respecter les différents contrôles définis via la nomenclature 378 et le paramétrage du type de carte.

Nom : charMax255
Description : Chaîne composée d'au plus 255 caractères.
Format : string
Longueur mini-maxi : 1 - 255
