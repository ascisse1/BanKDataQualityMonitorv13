Nom : amountCurrency
Description : Devise de montant.

Les devises sont définies dans les nomenclatures 002 (devises alphanumériques) et 005 (devises numériques).

Choix : Non
  • amountCurrency
    • currency : simpleCurrency
      • alphaCode : char3
      • numericCode : char3
      • designation : charMax30
    • numberOfDecimals : integerBetween0And4

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
