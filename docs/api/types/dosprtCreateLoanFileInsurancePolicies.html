Nom : subscriberNumber
Description : Numéro d¿adhérent (numéro de police).
Commentaire :

Si le Montant 7 du  paramètre ASSURANCE en nomenclature 098 est renseigné (valeur possible entre 4 et 15), alors le numéro de police est affecté automatiquement avec le type d'affectation auquel est concaténé un entier incrémenté à partir du dernier numéro de police inséré pour le même type d'affectation, le tout sur le nombre de caractères indiqué dans le Montant 7 du paramètre ASSURANCE.

Sinon, le numéro de police renseigné ne doit pas déjà exister dans la table des polices d’assurance.

Si le numéro de police n'est pas renseigné, il est affecté automatiquement à partir du Montant 1 du paramètre ASSURANCE en nomenclature 098 incrémenté de 1, et formaté sur 20 caractères.

Nom : bankPremium
Description : Prime banque.
Commentaire :

Si la prime banque n’est pas renseignée, elle est affectée avec la valeur paramétrée pour la condition BSUaaa (aaa = type d'affectation), ou ASBAN2 ou COnttt (n = numéro de poste de commission et ttt = type de prêt).

Si un barème est paramétré au niveau du type d'affectation de l'assurance, alors la prime banque, si elle n’est pas renseignée, est initialisée à partir de la valeur du barème.

La prime banque doit être comprise entre les bornes minimum et maximum paramétrées pour la condition BSUaaa (aaa = type d'affectation), ou ASBAN2 ou COnttt (n = numéro de poste de commission et ttt = type de prêt).

Nom : insurerPremium
Description : Prime assureur.

Si la prime assureur n’est pas renseignée, elle est affectée avec la valeur paramétrée pour la condition ASnaaa (n = numéro de poste de commission et aaa = type d'affectation) ou COnttt (n = numéro de poste de commission et ttt = type de prêt) ou COnPRT (n = numéro de poste de commission).

Si un barème est paramétré au niveau du type d'affectation de l'assurance, alors la prime assureur, si elle n’est pas renseignée, est initialisée à partir de la valeur du barème.

La prime assureur doit être comprise entre les bornes minimum et maximum paramétrées pour la condition ASnaaa (n = numéro de poste de commission  et aaa =  type d'affectation) ou COnttt (n = numéro de poste de commission et ttt =  type de prêt) ou COnPRT (n =  numéro de poste de commission).
