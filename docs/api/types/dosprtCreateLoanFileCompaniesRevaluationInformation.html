Nom : dosprtCreateLoanFileCompaniesRevaluationInformation
Description : Informations sur la réévaluation des crédits aux sociétés.

Pour un client de type société (2).

Choix : Non
  • dosprtCreateLoanFileCompaniesRevaluationInformation
    • rateTypeToBeApplieOnCapital : char3
    • previousRateMargin : decimal9_6

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : rateTypeToBeApplieOnCapital
Description : Type de taux à appliquer sur le capital lors de l'appel d'échéance.
Commentaire :

Liste des valeurs possibles :

  • M00 = Cours moyen
  • A00 = Cours achat
  • V00 = Cours vente
  • Clé existant en nomenclature 128

Valeur par défaut : typetaux

Nom : char3
Description : Type avec un nombre de caractères égal à 3.
Format : string
Longueur fixe : 3
Nom : previousRateMargin
Description : Marge à appliquer sur le taux précédent.
Commentaire :

Valeur par défaut : marge_txcap

Si la valeur par défaut n’est pas paramétrée alors les conditions MREttt (ttt correspond au code du type de prêt) ou MREPRT sont recherchées afin de déterminer la marge par défaut ainsi que les bornes minimale et maximale.

Si la valeur par défaut est paramétrée, les bornes minimale et maximale sont chacunes initialisées à  100.

La valeur de la marge renseignée doit être comprise dans ces bornes.

Nom : decimal9_6
Description : Type décimal à 9 chiffres dont 6 décimales.
Format : decimal
Nombre total de chiffres : 9
