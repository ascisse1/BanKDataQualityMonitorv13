  • ceilingType
Nom : ceilingType
Description : Ce type définit le type de limite utilisé (autorisation plafonnée, client ou groupe).
Format : string
Valeurs possibles :
G Limite Groupe
A Autorisation plafonnée
C Limite client

Liste de valeurs : A, C, G

A = Autorisation plafonnée       

C = Limite client                                                        

G = Limite Groupe
