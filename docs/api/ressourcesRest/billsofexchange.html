Service SOAP : getBillOfExchangeDetail
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/billsofexchange/v1/{billId}
Description : The service getBillOfExchangeDetail allows getting detailed information on a bill of exchange corresponding to a unique identifier received. For the selected bill of exchange, the following information is returned: - details of the bill for recovery, - details of taxes and bank charges for the bill, - the status of the linked standard bill of exchange (LCN).

Service SOAP : getBillOfExchangeList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/billsofexchange/v1/
Description : The service getBillOfExchangeList allows returning a list of bills matching the search criteria received. The search is based on the optional criteria below: - the branch (code and/or designation), - the institution code and/or branch sort code of the drawee, - the bill number, - the remittance slip number, - the code and/or the name to be displayed of the remitter customer, - the operation code (code and/or designation), - the remitter's account : branch (code and/or designation), currency (alphanumeric code and/or digital code and/or designation), number, suffix, - the bill currency (alphanumeric code and/or digital code and/or designation), - the drawee identifier: internal or other bank drawee code and/or customer code and/or name, - the drawee's account: account in our bank (branch (code and/or designation), currency (alphanumeric code and/or digital code and/or designation), number, suffix), or other bank account (bank, branch sort code, account number), - the code of the bill type - the linked event number, - the code of the linked bill type, - the current due date, - the presentation date, - the situation code of the bill after setting as unpaid, - the bill life code. For each bill returned, all data used in the search criteria will be indicated.
