Leasing files
  • leasings
    • v1
      • GETLIST
Service SOAP : getLeasingList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/leasings/v1/
Description : The service getLeasingList allows returning a list of leasing files matching the search criteria received. The search is based on the optional criteria below: - the leasing file identifier: branch (code and/or designation), file number, - the leasing type (code and/or designation), - the file currency (alphanumeric code and/or digital code and/or designation), - the file owner: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the free status (code and/or designation), - the imposed status, - the profitability date (or a date range). For each leasing file returned, all data used in the search criteria will be indicated as well as: - the simulation number, - the rental base amount.
