Service SOAP : getStoppageList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/stoppages/v1/
Description : The service getStoppageList allows returning a list of stoppages matching the search criteria received. The search is based on the optional criteria below: - the stoppage type (customer and/or account), - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the account: number (internal and/or IBAN and/or external format) and/or class and/or additional information (identifier and/or value (or value range) and/or designation), - the stoppage: code and/or designation, start date (or date range), end date (or date range), status, release date (or date range). For each stoppage returned, all data used in the search criteria will be indicated as well as: - the user who released the stoppage, - the user who created the stoppage, - the stoppage reason, - the stoppage release reason.
