Service SOAP : getStoredChequesList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/storedcheques/v1/
Description : The service getStoredChequesList allows returning a list of cheques in stock matching the search criteria received. The search is based on the optional criteria below: - the branch (code and/or designation), - the serial number, - the cheque number (or a number range), - the holder's account: number (internal and/or IBAN and/or external format) and/or class and/or customer (code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference) and/or additional information (identifier and/or value (or value range) and/or designation), - the cheque amount (or an amount range), - the blocked amount (or an amount range), - the situation of the cheque, - the cheque type, - the cheque situation date (or a date range), - the cheque book request number (or a number range), - the cheque book type. For each cheque returned, all data used in the search criteria will be indicated.
