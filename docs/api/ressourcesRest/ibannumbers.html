International Bank Account Numbers (IBAN)
  • ibannumbers
    • v1
      • GET
Service SOAP : getIBANDetail
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/ibannumbers/v1/{ibanId}
Description : This service allows the validation of an IBAN on the structure, and optionally, the elements composing it. The IBAN passed in request is checked in all cases on its structure (length and key). If the detail is requested, the validation of the IBAN is also carried out on the data concerning the BIC code and the branch code. For example, if an anomaly occurs on the BIC code, then the validation fails.
