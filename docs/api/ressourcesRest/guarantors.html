Service SOAP : getGuarantorList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/guarantors/v1/
Description : The service getGuarantorList allows returning the list of customers and/or third parties which are already guarantors or likely to become one corresponding the search criteria received. The search is based on the following optional criteria: - Information about the guarantor¿s customer, - Information about the guarantor¿s third party. For each returned guarantor, all data planned in the search criteria will be specified.
