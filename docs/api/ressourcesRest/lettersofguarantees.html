Service SOAP : getLGList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/lettersofguarantees/v1/
Description : The service getLGList allows returning a list of letters of guarantee matching the search criteria received. The search is based on the optional criteria below: - the L/G identifier: branch (code and/or designation), L/G number, - the currency (alphanumeric code and/or digital code and/or designation), - the L/G type (code and/or designation), - the file number, - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the beneficiary: type (customer/third party) and/or code and/or name to be displayed, - the value date (or a date range), - the expiry date (or a date range), - the L/G situation, - the bill reference. For each L/G returned, all data used in the search criteria will be indicated as well as: - the amount, - the commission rate, - the frequency.
