Service SOAP : getTypeOfTDList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/termdeposittypes/v1/
Description : The service getTypeOfTDList allows returning a list of TD types matching the search criteria received. The search is based on the optional criteria below: - the identifier of the type: branch (code and/or designation), TD type (code and/or designation), - the operation code (code and/or designation), - the TD group, - the product (code and/or designation and/or attribute). For each TD type returned, all data used in the search criteria will be indicated as well as: - the duration unit, - the duration.
