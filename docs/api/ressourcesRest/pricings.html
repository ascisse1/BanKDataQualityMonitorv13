Product or package pricings
  • pricings
    • v1
      • GETLIST
Service SOAP : getPricingDetail
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/pricings/v1/
Description : The service getPricingDetail allows listing detailed information of product or package pricing corresponding to a single identifier received. The mandatory identifier received contains: - Pricing code, - Pricing type, - Package code, - Product code. Information sent back by the service is: - The identifier received, completed by the possible designation, - Pricing characteristics (base pricing, pricing mode, reduction percentage, triggering method, operation, quantity, amount, account nature), - Information about the first triggering, - Information about the following triggering, - Information about closing of invoice at maturity, - Implementation charges, - Preferential pricing, - Application start date of pricing, - Application end date of pricing, - Account activity, - Information of subservices.
