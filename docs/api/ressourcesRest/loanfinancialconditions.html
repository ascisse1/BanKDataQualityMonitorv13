Financial conditions that can be applied to a loan file
  • loanfinancialconditions
    • v1
      • GETLIST
Service SOAP : getAmortizableLoanFinancialConditionsDetail
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/loanfinancialconditions/v1/
Description : The service getAmortizableLoanFinancialConditionsDetail allows finding detailed information about financial conditions that can be applied to a loan file corresponding to a single identifier received. Information sent back is linked to conditions on: - Capital amount - Number of instalments, - Interest rate, - Commissions, - Charges, - Tax deduction, - Retention money.
