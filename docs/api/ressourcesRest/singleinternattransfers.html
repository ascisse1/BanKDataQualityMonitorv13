Service SOAP : createSingleInternationalTransfer
Code retour succes : 201 - Created
Uri : http://soprabanking.com/amplitude/singleinternattransfers/v1/
Description : The API createSingleInternationalTransfer allows to create an international transfer (SWIFT) between two accounts to a bank located abroad with an amount and a reason for transfer. The API will return the status of the transfer and indicate whether the transfer has been created or not, and the reason for rejection encountered when the transfer wasn¿t created. Here is the list of the different statuses: ACTC : Transfer correctly integrated but not processed ACSP : Integrated and properly processed transfer PDNG : Transfer processed but with an error (The error encountered will be described in the response) RJCT : Transfer rejected
