Service SOAP : getCoDDetail
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/certificatesofdeposit/v1/{codId}
Description : The service getCoDDetail allows getting detailed information on a certificate of deposit corresponding to a unique identifier received. For the selected certificate of deposit, the following information is returned: - details of the certificate of deposit, - the list of concerned accounts, - the situation in file currency, - the situation in domestic currency.

Service SOAP : getCoDList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/certificatesofdeposit/v1/
Description : The service getCoDList allows returning a list of certificates of deposit matching the search criteria received. The search is based on the optional criteria below: - the identifier of the certificate of deposit: branch (code and/or designation), CD subscription number, CD number, - the currency (alphanumeric code and/or digital code and/or designation), - the type of certificate of deposit (code and/or designation), - the file account: number (internal and/or IBAN and/or external format) and/or class and/or customer and/or additional information (identifier and/or value (or value range) and/or designation), - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the pledge indicator, - the naming indicator, - the value date (or a date range), - the maturity date (or a date range), - the CD situation. For each CD returned, all data used in the search criteria will be indicated as well as: - the execution date, - the initial amount, - the amount remaining to be reimbursed, - the rate, - the capitalisation indicator.

Description : Simulations
Service SOAP : getCoDSimulationDetail
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/certificatesofdeposit/v1/{codId}/simulations
Description : The service getCoDSimulationDetail allows simulating the calculation of interest for a certificate of deposit either corresponding to a unique identifier received or matching the detailed CD information received. For the selected or simulated CD, the following information is returned: - the simulation of interest for all the CD duration, - the simulation of interest on maturity date, - the simulation of interest in case of breaking, - the simulation of interest for the next period, - the simulation of interest advance.
