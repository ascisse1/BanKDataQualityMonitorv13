Service SOAP : getCardList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/cards/v1/
Description : The service getCardList allows returning a list of cards matching the search criteria received. The search is based on the optional criteria below: - the card branch (code and/or designation), - the input branch (code and/or designation), - the delivery branch (code and/or designation), - the account number (internal and/or IBAN and/or external format), - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the holder: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the card type (code and/or designation), - the card reference, - the card number, - the truncated card number, - the card situation, - the PIN code situation, - the date (or a date range) of the card remanufacturing batch, - the date (of a date range) of the PIN code manufacturing batch, - the card expiry date (of a date range), - the card stoppage reason, - the card cancellation reason. For each card returned, all data used in the search criteria will be indicated as well as: - the embossing, - the corporate name, - the holder type.
