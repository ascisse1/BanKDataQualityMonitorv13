Service SOAP : createScoringFile
Code retour succes : 201 - Created
Uri : http://soprabanking.com/amplitude/scoringfiles/v1/
Description : The service createScoringFile allows creating a scoring file establishing a decision (approval or rejection) according to an overall score calculated. The score calculated, which allows to decide (Approval or Rejection of loan request) is based on a scoring model (code and version), as well as the following information: - Customer code for which the score calculation has to be done (mandatory if the scoring model involves at least one customer information) - External file reference, - List of values of ¿external scoring data¿ The service creates a score file and supplies, among other things, apart from a score and a scoring decision, the reference of the file created.

400 Bad Request
