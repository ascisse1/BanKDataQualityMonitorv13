Service SOAP : cancelFundReservation
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/fundreservations/v1/{reservationId}
Description : The service cancelFundReservation allows releasing a reservation of funds of nature RESERV corresponding to a unique identifier received. The service returns the balance available on the account after completion of the operation.

Service SOAP : createFundReservation
Code retour succes : 201 - Created
Uri : http://soprabanking.com/amplitude/fundreservations/v1/
Description : The service createFundReservation allows creating a reservation of funds of nature RESERV. The characteristics information on the reservation of funds is required. The service returns: - the identifier of the reservation of funds created: branch (code and designation), operation (code and designation), event number, transaction nature, - the status, - the disapproval list (code and designation ; 5 max.), - the balance available on the account after completion of the operation.


Codes Erreur :
400 Bad Request
Service SOAP : getFundReservationList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/fundreservations/v1/
Description : The service getFundReservationList allows returning a list of reservations of funds matching the search criteria received. The search is based on the optional criteria below: - the identifier of the reservation of funds: branch (code and/or designation), operation code (code and/or designation), event number, transaction nature, - the status, - the reason (code and/or designation), - the amount (or an amount range), - the expiry date (or a date range), - the creation date (or a date range, - the file account: number (internal and/or IBAN and/or external format) and/or class and/or customer (code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference) and/or additional information (identifier and/or value (or value range) and/or designation). For each reservation of funds returned, all data used in the search criteria will be indicated.
