Service SOAP : getCustomerGroupList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/customergroups/v1/
Description : The service getCustomerGroupList allows returning a list of customer group matching the search criteria received. The search is based on the optional criteria below: - the customer's group (code and/or name of the group) - the customer or third party (code and/or name of the customer or third party) For each customer group returned, all data used in the search criteria will be indicated.
