Service SOAP : getAlertList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/alerts/v1/{alertId}
Description : The service getAlertList allows returning a list of alerts matching the search criteria received. The search is based on the optional criteria below: - the alert number, - the alert: code, designation, type, distribution channel (code and/or designation), field, - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the alert status, - the seriousness level. For each alert returned, all data used in the search criteria will be indicated as well as: - the alert reference, - the reference date, - the amount and its currency, - the account officer user code, - the alert date, - the code of the user who initiated the alert.
