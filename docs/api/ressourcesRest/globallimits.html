Service SOAP : getGlobalLimitList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/globallimits/v1/
Description : The service getGlobalLimitList allows returning a list of limits and ceilings matching the search criteria received. The search is based on the optional criteria below: - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the global line identifier (reference and/or designation), - the global line type (code and/or designation), - the customer global line indicator, - the account linking, - the start date (or a date range), - the expiry date (or a date range), - the amount (or an amount range), - the balance merge indicator. For each global line returned, all data used in the search criteria will be indicated.
