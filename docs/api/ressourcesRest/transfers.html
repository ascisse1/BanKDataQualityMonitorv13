Service SOAP : getDomesticTransferList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/transfers/v1/
Description : The service getDomesticTransferList allows returning a list of local transfers matching the search criteria received. The search is based on the optional criteria below: - the branch (code and/or designation), - the operation (code and/or designation), - the event number, - the local transfer type (code and/or designation), - the local transfer status, - the local transfer amount (or amount range), - the local transfer currency (alphanumeric code and/or digital code and/or designation), - the local transfer issuer: account number (internal and/or IBAN and/or external format) and/or class and/or customer (populationFile) and/or additional account information (identifier and/or value (or value range) and/or designation), - the local transfer beneficiary: account number in internal and/or IBAN and/or external format, or other bank format, or international format, - the event date (or a date range), - the local transfer execution date (or a date range), - the local transfer execution day, - the local transfer frequency, - the type and/or reference of the local transfer and/or the SEPA file reference. For each local transfer returned, all data used in the search criteria will be indicated as well as: - the local transfer suspension start date, - the local transfer suspension end date, - the transaction nature, - the reference amount, - the reference currency (alphanumeric code, digital code, designation), - the indicator of resident debited person, - the indicator of resident credited person, - the economic code.
