Service SOAP : getBillOfExchangeOtherBankList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/billsofexchangeotherbank/v1/
Description : The service getBillOfExchangeOtherBankList allows returning a list of bills of exchange issued from other banks corresponding to the search criteria received. The search is based on the optional criteria below: - the branch of the bill (code and/or designation), - the source branch (code and/or designation), - the drawee's bank code and/or branch sort code, - the presentation bank code and/or branch sort code, - the bill number, - the bill currency (alphanumeric code and/or digital code and/or designation), - the drawee identifier: internal or other bank drawee code and/or customer code and/or name, - the drawee's account in another bank: bank, branch sort code, account number, - the code of the bill type, - the linked event number, - the code of the linked bill type, - the presentation date. For each bill returned, all data used in the search criteria will be indicated.
