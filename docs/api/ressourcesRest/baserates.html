Service SOAP : getBaseRateList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/baserates/v1/
Description : The service getBaseRateList allows returning a list of base rates matching the search criteria received. The search is based on the optional criteria below: - the base rate (code and/or designation), - the base rate validity date (or a date range). For each base rate returned, all data used in the search criteria will be indicated as well as the base rate value.
