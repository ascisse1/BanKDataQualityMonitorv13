Service SOAP : createStandardBillOfExchangeRequest
Code retour succes : 201 - Created
Uri : http://soprabanking.com/amplitude/standardbillsofexchange/v1/
Description : The service createStandardBillOfExchangeRequest allows creating a request for LCN (standard bills of exchange) books. The characteristics of the request are required: branch, account, book type, number of books, availability method. They can come along with other optional information: - the characteristics of the request fees (amount and value date), - the characteristics of the Powers of Attorney (branch and proxy identifier). The service returns: - the request number, - the disapproval list (code and designation).


Codes Erreur :
400 Bad Request
Service SOAP : getStandardBillOfExchangeRequestList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/standardbillsofexchange/v1/
Description : The service getStandardBillOfExchangeRequestList allows returning a list of requests for standard of exchange (LCN) books matching the search criteria received. The search is based on the optional criteria below: - the branch (code and/or designation), - the request number, - the account number (internal and/or IBAN and/or external format), - the customer : code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the standard bill of exchange book dispatch method, - the request date (or a date range), - the number (or a number range)of books requested, - the customisation date (or a date range), - the number (or a number range) of customised books, - the remittance date (or a date range), - the number (or a number range) of books remitted, - the destruction date (or a date range), - the number (or a number range) of books destroyed, - the order date (or a date range), - the number (or a number range) of books ordered, - the last modification date (or a date range), - the situation of the request, - the status of the request. For each bill returned, all data used in the search criteria will be indicated.
