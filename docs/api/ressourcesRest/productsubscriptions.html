Service SOAP : createProductSubscription
Code retour succes : 201 - Created
Uri : http://soprabanking.com/amplitude/productsubscriptions/v1/
Description : The service createProductSubscription allows creating a product subscription. The reference of the product subscription to be created is either provided or automatically allocated. The customer code, the product code and the subscription status are required. This information can come along with other optional information: - the code and reference of package subscription, - the file type and/or number, - the identifier of the main account in internal format, - the subscription end date, - the information on product pricing (code, type, invoicing account identifier in internal format, quantity of subscribed products, indicator of invoicing of implementation fees, amount without taxes at implementation time), - a comment, - an indicator of recovery of unpaid amount. The service returns the reference of the product subscription created.


Codes Erreur :

Service SOAP : getProductSubscriptionList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/productsubscriptions/v1/
Description : The service getProductSubscriptionList allows returning a list of subscribed products or packages matching the search criteria received. The search is based on the optional criteria below: - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the product (code and/or designation and/or attribute), - the package (code and/or designation), - the module of the linked file (code and/or designation), - the account: number (internal and/or IBAN and/or external format) and/or class and/or customer and/or additional information (identifier and/or value or value range and/or designation), - the processing code. For each subscribed product or package returned, all data used in the search criteria will be indicated as well as: - the file type and number, - the subscription reference, - the subscription start and end dates .
