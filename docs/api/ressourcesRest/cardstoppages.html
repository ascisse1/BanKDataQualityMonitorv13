Service SOAP : cancelCardStoppage
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/cardstoppages/v1/{cardId}
Description : The service cancelCardStoppage allows releasing a stoppage on the card corresponding to a unique identifier received. The stoppage release type is required. It can come along with comments that will be stored in a history table.

Code retour succes : 201 - Created
Uri : http://soprabanking.com/amplitude/cardstoppages/v1/
