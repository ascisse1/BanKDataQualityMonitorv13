Programme Amplitude : abwppedatc
Service générique : createPEPData


Programme de création des informations PPE d'un client ou d'un tiers.

 

Concernant le traitement fonctionnel et les règles de gestion, se référer aux documents utilisateurs :

0260-2-11.XModule Lutte Anti-Blanchiment

0260-3-11.XModule Lutte Anti-Blanchiment

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
