Tables amplitude :

Ce service permet d'obtenir le détail d'une tarification correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques d'une tarification sont renvoyées.

Commentaire :

la valeur de l’élément, la sélection sur le produit sera différente :

  • Si l’élément vaut SERVICE PRODUCT : seuls les produits vérifiant bkprod.att différent de 01 seront sélectionnés.
  • Si l’élément vaut ACCOUNT PRODUCT : seuls les produits vérifiant bkprod.att égal à 01 seront sélectionnés.
  • Si l’élément vaut FILE PRODUCT : seuls les produits vérifiant bkprod.dos égal à O seront sélectionnés.
