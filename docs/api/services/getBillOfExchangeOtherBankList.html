Programme Amplitude : guweffefar
Service générique : getBillOfExchangeOtherBankList

Tables amplitude :
  • bkeve

Ce service permet de lister les effets provenant d'autres banques, correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques d'un effet provenant d'autres banques sont renvoyées.

Afin de limiter le nombre d’enregistrements sélectionnés et d’éviter le déchargement d’une table dans son intégralité, plusieurs combinaisons de critères obligatoires sont définies. Au moins l’une de ces combinaisons doit être donnée en entrée au service pour permettre une restitution de résultats.

Le tableau ci-dessous liste ces combinaisons, où une ligne contient une combinaison possible :

Combinaison de critères

Etablissement et guichet du tiré (Balises branch et cashRegister de l’élément draweeEstablishment)

Compte du tiré (Balises establishment, cashDesk et accountNumber de l’élément draweeAccountOtherBank)
