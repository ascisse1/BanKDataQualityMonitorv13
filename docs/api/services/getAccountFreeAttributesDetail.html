Programme Amplitude : cbwcptatld
Service générique : getAccountFreeAttributesDetail

Tables amplitude :
  • bkcom

Ce service permet de lister les attributs libres d'un compte correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques des attributs libres d'un compte sont renvoyées.
