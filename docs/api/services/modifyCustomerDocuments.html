Programme Amplitude : cbwclidocm
Service générique : modifyCustomerDocuments


Ce service permet de modifier (par annule et remplace) l'ensemble des documents d'un client correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit après avoir vérifié leur cohérence.

/!\ Il est interdit de modifier une fiche client en attente de validation.

 

Concernant le traitement fonctionnel et les règles de gestion, se référer aux documents utilisateurs :

0020-3-11.XGestion bancaire - Référentiel - TOME 2

Commentaire :

Le code client n’est pas obligatoire si le service de modification des documents d’un client est appelé par le service createCustomer, et est obligatoire dans tous les autres cas.
