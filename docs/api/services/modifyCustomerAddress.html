Service générique : modifyCustomerAddress

Tables amplitude :
  • bkadcli

Ce service permet de modifier une adresse d'un client correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit après avoir vérifié leur cohérence et géré d¿éventuelles valeurs par défaut.

/!\ Il est interdit de modifier une fiche client en attente de validation.

Les contrôles relatifs à une adresse au format géographique reposent sur le paramétrage en nomenclature 098 du paramètre FMT-ADRGEO. Si le paramètre n'existe pas, alors les valeurs suivantes sont utilisées par défaut :
