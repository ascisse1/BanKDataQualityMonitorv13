Service générique : getDocumentList

Tables amplitude :
  • bkcli

Ce service permet de lister les documents correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques des documents sont renvoyées.

Afin de limiter le nombre d’enregistrements sélectionnés et d’éviter le déchargement d’une table dans son intégralité, plusieurs combinaisons de critères obligatoires sont définies. Au moins l’une de ces combinaisons doit être donnée en entrée au service pour permettre une restitution de résultats.

Le tableau ci-dessous liste ces combinaisons, où une ligne contient une combinaison possible :

Code de la nature du document (Balise code contenue dans le type documentNature)

Libellé de la nature du document (Balise designation contenue dans le type documentNature)
