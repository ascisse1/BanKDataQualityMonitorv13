Programme Amplitude : cbwcomresc
Service générique : createFundReservation

Tables amplitude :
  • bkreserv
  • bkope
  • bkcom

Ce service permet la création d'une réservation de fonds de nature RESERV à l'image du programme de saisie gureserv. Le code opération doit être paramétré pour la nature RESER2 en nomenclature 052. Le service vérifie la disponibilité du Core Banking, et est disponible 24/24, excepté pendant une période du traitement de fin de journée durant laquelle les soldes des comptes ne sont pas disponibles et il n'est pas possible d'effectuer de transaction.
