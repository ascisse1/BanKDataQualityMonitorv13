Service générique : getAutomaticDirectDebitListV2


Ce service permet de lister les prélèvements automatiques émis, qu'ils soient SEPA ou non SEPA, correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques d'un prélèvement sont renvoyées.

 

getAutomaticDirectDebitListV2
Nature Elément Détail
Modification getAutomaticDirectDebitListRequest.sepaOrNotSepaDirectDebitIdentifier.notSepaDirectDebitIdentifier.sequenceNumber Passage de 5 caractères à 6 caractères.

 

Afin de limiter le nombre d’enregistrements sélectionnés et d’éviter le déchargement d’une table dans son intégralité, plusieurs combinaisons de critères obligatoires sont définies. Au moins l’une de ces combinaisons doit être donnée en entrée au service pour permettre une restitution de résultats.
