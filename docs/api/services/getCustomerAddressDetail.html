Service générique : getCustomerAddressDetail

Tables amplitude :
  • evlang
  • bkadcli

Ce service permet de lister les adresses des clients correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque adresse de client sélectionné, l'identifiant du client, le code langue de restitution de l'adresse, le format de l'adresse, ainsi qu'un ensemble de données liées à l'adresse du client sont renvoyés.
