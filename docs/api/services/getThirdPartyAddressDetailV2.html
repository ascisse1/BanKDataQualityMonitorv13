Programme Amplitude : cbwtieadrdv2
Service générique : getThirdPartyAddressDetailV2


Ce service permet de fournir le détail d'une ou de toutes les adresses d'un tiers correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit.
