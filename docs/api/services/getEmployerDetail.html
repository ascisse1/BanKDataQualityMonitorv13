Service générique : getEmployerDetail

Tables amplitude :
  • bkcli
  • bkempl
  • evlang
  • bkdempl

Ce service permet d'obtenir le détail d'un employeur correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques d'un employeur sont renvoyées. Pour un employeur de type client ou tiers, les informations de contact ne sont pas restituées.
