Service générique : getCustomerEmailAddressList

Tables amplitude :
  • bkcli
  • bkemacli

Ce service permet de lister les adresses email d'un client correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque email sélectionné, le nom du client, le type d¿email (personnel, professionnel¿) ainsi qu'un ensemble de données complémentaires sont renvoyés.

 

Afin de limiter le nombre d’enregistrements sélectionnés et d’éviter le déchargement d’une table dans son intégralité, plusieurs combinaisons de critères obligatoires sont définies. Au moins l’une de ces combinaisons doit être donnée en entrée au service pour permettre une restitution de résultats.

Le tableau ci-dessous liste ces combinaisons, où une ligne contient une combinaison possible :

Combinaison de critères

Client (Balise customerNumber contenue dans le type restrictedCustomer)

Type de téléphone (Balise code contenue dans le type mailType)
