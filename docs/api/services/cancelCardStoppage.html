  • cancelCardStoppage
Programme Amplitude : mowmonopps
Service générique : cancelCardStoppage

Tables amplitude :
  • mocarte_crypt
  • mocarte_data
  • bkcom

Ce service permet de lever l'opposition sur une carte selon les critères de sélection contenus dans le flux XML qu'il reçoit.

Pendant la période d’indisponibilité du Core Banking Amplitude, le service est momentanément inaccessible.

La prise de frais sur la levée d’une opposition génère un événement soit sur la date comptable si le site central est ouvert, soit sur le prochain jour ouvré si le site central est fermé.

Dans le cas d’une prise de frais, le solde indicatif du compte est imputé directement en journée, et seulement après TFJ pendant la nuit.
