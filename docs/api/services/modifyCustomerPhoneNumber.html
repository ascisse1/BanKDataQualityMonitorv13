Service générique : modifyCustomerPhoneNumber

Tables amplitude :
  • bktelcli

Ce service permet de modifier un numéro de téléphone d'un client, correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit après avoir vérifié leur cohérence et géré d¿éventuelles valeurs par défaut.

/!\ Il est interdit de modifier une fiche client en attente de validation.

Nom : phoneNumber
Description : Numéro de téléphone.
Commentaire :

La valeur du numéro de téléphone est contrôlée à partir de son format s’il est valorisé.

Le numéro de téléphone est alors complété avec l’indicatif national paramétré en nomenclature 199 dans le Libellé 3 en appliquant le masque paramétré dans le Libellé 4 (i représentant l’indicatif et n un chiffre du numéro de téléphone) puis amputé du premier zéro du numéro (situé juste après l’indicatif national) si le Taux 1 vaut 1.

La longueur du numéro de téléphone après l’ajout de l’indicatif national et la suppression du premier zéro éventuel doit correspondre à la longueur paramétrée dans le Montant 1.
