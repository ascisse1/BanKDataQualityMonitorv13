Programme Amplitude : cbwclibudm
Service générique : modifyCustomerBudget


Ce service permet de modifier le budget d'un client ainsi que les revenus et charges associés à ce budget (par annule et remplace), correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit après avoir vérifié leur cohérence.

/!\ Il est interdit de modifier une fiche client en attente de validation.

 

Concernant le traitement fonctionnel et les règles de gestion, se référer aux documents utilisateurs :

0020-3-11.XGestion bancaire - Référentiel - TOME 2
