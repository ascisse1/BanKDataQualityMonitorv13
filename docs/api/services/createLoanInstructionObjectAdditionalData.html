Service générique : createLoanInstructionObjectAdditionalData


Le service createLoanInstructionObjectAdditionalData permet de valoriser une donnée complémentaire dans l'objet d'un projet de financement. Seule une donnée complémentaire peut être envoyée à la fois. La nature de la donnée complémentaire à valoriser doit être autorisée pour le type de dossier. Le format choisi doit être compatible avec celui défini pour la nature de donnée complémentaire.
