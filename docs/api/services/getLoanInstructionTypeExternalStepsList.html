Programme Amplitude : icwidceter
Service générique : getLoanInstructionTypeExternalStepsList


Le service getLoanInstructionTypeExternalStepsList permet de retourner la liste des étapes à traiter par une procédure externe pour un type de financement. Dans le détail de chaque étape retournée parmi la liste, on retrouve code et libellé de l'étape, les décisions possibles, des informations sur les niveaux de décision, la liste des réserves et motifs de rejets possibles.
