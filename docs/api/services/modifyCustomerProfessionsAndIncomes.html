Programme Amplitude : cbwcliprfm
Service générique : modifyCustomerProfessionsAndIncomes


Ce service permet de modifier (par annule et remplace) l'ensemble des professions et revenus d'un client correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit, après avoir vérifié leur cohérence.

/!\ Il est interdit de modifier une fiche client en attente de validation.

 

Concernant le traitement fonctionnel et les règles de gestion, se référer aux documents utilisateurs :

0020-3-11.XGestion bancaire - Référentiel - TOME 2

Commentaire :

Si un nombre de jours est paramétré dans le Montant 1 de la nomenclature 098 pour le paramètre CONTRDEMB, alors la date d’embauche doit être inférieure ou égale à la date comptable à laquelle on ajoute ce nombre de jours.

La date d’embauche doit être supérieure à la date de naissance du client.
