Programme Amplitude : cbwbksbnkdv2
Service générique : getBankDetailV2


Ce service permet de restituer le détail de la banque correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour la banque sélectionnée, les informations complémentaires comme la liste des comptes et les adresses de la banque sont renvoyées.
