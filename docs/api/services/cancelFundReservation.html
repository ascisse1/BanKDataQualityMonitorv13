Programme Amplitude : cbwcomress
Service générique : cancelFundReservation


Ce service permet la levée d'une réservation de fonds de nature RESERV correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques de la réservation de fonds éventuellement supprimée sont renvoyées.

 

Concernant le traitement fonctionnel et les règles de gestion, se référer aux documents utilisateurs :
