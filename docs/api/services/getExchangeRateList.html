Programme Amplitude : cbwtxclisr
Service générique : getExchangeRateList


Ce service permet de lister les taux de change correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques des taux de change sont renvoyées.

      • getExchangeRateListRequest : getExchangeRateListRequest

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Commentaire :

Si cette balise est laissée vide, alors la recherche des taux se fera sur toutes les natures possibles définies dans le type "rateSearchNatureType".
Pour les taux étrangers, la nature de recherche F n'existe pas, il est donc interdit de la demander et elle n'est pas considérée dans le cas d'une recherche sur toutes les natures possibles.
