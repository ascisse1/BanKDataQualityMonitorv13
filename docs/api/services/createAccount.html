Ce service permet de créer un compte rattaché à un client, un dossier ou un produit, correspondant aux critères contenus dans le flux XML qu'il reçoit. Le cas d'un compte produit est traité sans prendre en compte toutes les tables annexes du compte produit (adresses, téléphones, contacts, emails...).

Le service fait appel à la fonction métier de création de compte, pour un compte dossier ou produit.

Dans le cas d'une création de compte produit, les valeurs par défaut associées au produit sont appliquées.

S'il existe une opposition sur le client du compte alors une opposition est positionnée sur le compte créé.

S'il s'agit d'une création de compte rattaché à un produit d'attribut "compte", la souscription du produit est enregistrée après avoir contrôlé l'éligibilité du client à la souscription.

Ce service ne permet ni la mise en instance de fermeture, ni la fermeture d’un compte, car cela demande une interaction obligatoire avec l’utilisateur afin de lui notifier tous les dossiers actifs qui portent sur le compte, pour qu’il puisse ensuite valider ou abandonner son action.

Ce service n'est pas autorisé dans le cas où le système de numérotation de compte implique une demande de saisie manuelle et partielle du numéro de compte.
