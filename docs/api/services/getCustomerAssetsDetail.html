Programme Amplitude : cbwclipatd
Service générique : getCustomerAssetsDetail


Ce service permet de fournir le détail des patrimoines d'un client correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque détail d'en-tête de patrimoine de client sélectionné, le code du client, l'année de référence du patrimoine, le niveau de fiabilité du patrimoine, l'exhaustivité du patrimoine ainsi que le détail des patrimoines du client sont renvoyés.
