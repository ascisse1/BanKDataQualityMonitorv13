Service générique : modifyAccountV2


Ce service permet de modifier un compte rattaché à un client correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Ce service permet notamment de mettre à jour pour ce compte de clientèle, les adresses, les numéros de téléphones, les adresses email, les données complémentaires, les mémos et les contacts. Chaque modification est tracée dans la table de compte-rendu des modifications sur les comptes, en stockant la valeur avant et après mise à jour.

Remarque : Ce service ne permet pas de supprimer l’intégralité des adresses, des numéros de téléphones, des adresses email, des données complémentaires, des mémos et des contacts.

Ce service ne permet ni la mise en instance de fermeture ni la fermeture d’un compte car cela demande une intéraction obligatoire avec l’utilisateur, afin de lui notifier tous les dossiers actifs qui portent sur le compte, pour qu’il puisse ensuite valider ou abandonner son action.
