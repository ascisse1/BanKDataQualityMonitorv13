Programme Amplitude : dtwdatttdr
Service générique : getTypeOfTDList

Tables amplitude :
  • bkprod
  • bkope
  • bktydat

Ce service permet de lister les types de dépôts à terme correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques d'un type de dépôt à terme sont renvoyées.

Afin de limiter le nombre d'enregistrements sélectionnés et d'éviter le déchargement d'une table dans son intégralité, plusieurs combinaisons de critères obligatoires sont définies. Au moins l'une de ces combinaisons doit être donnée en entrée au service pour permettre une restitution de résultats.
Le tableau ci-dessous liste ces combinaisons, où une ligne contient une combinaison possible :

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.
