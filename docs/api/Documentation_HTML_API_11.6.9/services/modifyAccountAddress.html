Programme Amplitude : cbwcptadrm
Service générique : modifyAccountAddress

Tables amplitude :
  • bkcom

Ce service permet de modifier une adresse liée à un compte correspondant aux données contenues dans le flux XML qu'il reçoit après avoir vérifié leur cohérence. Chaque modification est tracée dans la table de compte-rendu des modifications sur les comptes, en stockant la valeur avant et après mise à jour.

Le service modifie uniquement une adresse du compte (C) ou une adresse du chéquier de type autre adresse (CH).

 

Concernant le traitement fonctionnel et les règles de gestion, se référer aux documents utilisateurs :
