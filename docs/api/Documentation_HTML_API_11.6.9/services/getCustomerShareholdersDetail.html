Service générique : getCustomerShareholdersDetail

Tables amplitude :
  • bkcli

Ce service permet de lister les actionnaires d'un client correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques d'un actionnaire sont renvoyées.

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.
