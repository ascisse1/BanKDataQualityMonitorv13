Ce service permet de lister les relations d'un client correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques des relations d'un client sont renvoyées.

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
