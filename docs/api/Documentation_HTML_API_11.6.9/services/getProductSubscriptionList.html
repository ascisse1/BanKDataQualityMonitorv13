Service générique : getProductSubscriptionList

Tables amplitude :

  • bkprod

Ce service permet de lister les produits et les packages souscrits correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques des produits et packages souscrits sont renvoyées.

Afin de limiter le nombre d’enregistrements sélectionnés et d’éviter le déchargement d’une table dans son intégralité, plusieurs combinaisons de critères obligatoires sont définies. Au moins l’une de ces combinaisons doit être donnée en entrée au service pour permettre une restitution de résultats.

Le tableau ci-dessous liste ces combinaisons, où une ligne contient une combinaison possible :

Combinaison de critères

Client (Balise customerNumber contenue dans le type populationFile)

Produit (Balise code contenue dans le type product)

Package (Balise code contenue dans le type package)
