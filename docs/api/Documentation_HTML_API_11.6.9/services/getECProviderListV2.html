  • getECProviderListV2
Programme Amplitude : ecwpvdrv2
Service générique : getECProviderListV2


Ce service permet de lister les fournisseurs de services correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques d'un fournisseur sont renvoyées.
