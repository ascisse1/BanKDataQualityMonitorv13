Service générique : createStandardBillOfExchangeRequest

Tables amplitude :
  • bkcom

Ce service permet de créer une demande de carnets de LCN correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit si les contrôles métiers le permettent.

 

Concernant le traitement fonctionnel et les règles de gestion, se référer aux documents utilisateurs :

0257-3-11.XCarnets de LCN

0258-3-11.XLettres de change normalisées (Maroc)

X : version mineure correspondant à la documentation.
