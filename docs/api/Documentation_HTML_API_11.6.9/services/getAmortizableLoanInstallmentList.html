Programme Amplitude : pswcdtechr
Service générique : getAmortizableLoanInstallmentList

Tables amplitude :
  • bkdosprt

Ce service permet de lister les échéances de dossiers de crédits amortissables correspondant aux critères de sélection contenus dans le flux XML qu¿il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques d¿un échéancier sont renvoyées.

Afin de limiter le nombre d’enregistrements sélectionnés et d’éviter le déchargement d’une table dans son intégralité, plusieurs combinaisons de critères obligatoires sont définies. Au moins l’une de ces combinaisons doit être donnée en entrée au service pour permettre une restitution de résultats.

Le tableau ci-dessous liste ces combinaisons, où une ligne contient une combinaison possible :

Combinaison de critères

Identifiant du dossier (Balises fileNumber, branch.code et amendmentNumber du type amortizableLoanIdentifier)

Date d’échéance (Balise date1 de l’élément installmentDate)

Etat de l’échéance (Balise installmentStatus)
