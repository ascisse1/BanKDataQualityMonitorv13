Ce service permet de créer un credoc import en pré-saisie correspondant aux critères contenus dans le flux XML qu'il reçoit.

Le type de credoc sur lequel est basé la pré-saisie est défini dans le Libellé 2 du paramètre CDI-E-BANK de la nomenclature 098.

 

Concernant le traitement fonctionnel et les règles de gestion, se référer aux documents utilisateurs :

0501-3-11.XCrédits Documentaires
