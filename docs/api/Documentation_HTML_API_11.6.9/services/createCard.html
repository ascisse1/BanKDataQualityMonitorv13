Programme Amplitude : mowmongesc
Service générique : createCard

Tables amplitude :
  • mocarte_crypt
  • mocarte_data
  • bkcom

Ce service permet de créer une carte correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques d'une carte sont renvoyées.

 

Concernant le traitement fonctionnel et les règles de gestion, se référer aux documents utilisateurs :

0200MON-V11.X-Gestion des cartes standard (Guide utilisateur)

0201MON-V11.X-Gestion des cartes standard (Paramétrage)

X : version mineure correspondant à la documentation.
