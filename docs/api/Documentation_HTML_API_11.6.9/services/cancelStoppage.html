Programme Amplitude : cbwopps
Service générique : cancelStoppage

Tables amplitude :
  • bkcom

Ce service permet de levée une opposition sur client ou compte correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit. Pour chaque enregistrement sélectionné, les données caractéristiques de levée une opposition sur client ou compte sont mises à jour.

  • cancelStoppage
    • StoppageRequest
      • requestHeader
        • requestId1
          • requestId : charMax50
        • serviceName : charMax50
        • timestamp : dateTime
        • originalName : charMax50
        • originalId : charMax50
        • originalTimestamp : dateTime
        • languageCode : languageCode
        • userCode : charMax10
        • channelCode : channelCode
      • cancelStoppageRequest : cancelStoppageRequest
        • [C] customerOrAccount : customerOrAccount
          • customer : customerCode
          • account : internalAccountKey
            • branch : char5
            • currency : currencyCode
            • account : charMax11
            • suffix : charMax2
        • stoppageCode : stoppageCode
        • stoppageEndReason : charMax40
    • StoppageResponse
      • responseHeader
        • requestId1
          • requestId : charMax50
        • responseId : charMax50
        • timestamp : dateTime
        • serviceVersion : charMax10
        • language : language
          • code : char3
          • designation : charMax30
      • responseStatus
        • statusCode : statusCodeType
        • messages : responseStatusMessages
          • [M] message : responseStatusMessage
            • nature : responseMessageNature
            • code : char6
            • [M] line : string
            • errorInformation : errorInformation
              • errorCode : string
              • convertedCode : string
              • errorMessage : string

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
