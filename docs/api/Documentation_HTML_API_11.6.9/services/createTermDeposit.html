Programme Amplitude : dtwdatgenc
Service générique : createTermDeposit

Tables amplitude :
  • bkcom

Ce service permet la création d'un DAT, des comptes, évènements et produits afférents à l'image du programme de saisie dtsaidat. Les opérations, natures d'opérations et types doivent être paramétrés en gestions des types, opérations et nomenclatures. Le service effectue les contrôles usuels de désaccord et de cohérence des données (valeurs par défaut, etc ). Il reçoit un flux XML contenant l'ensemble des informations caractéristiques d'un DAT.
