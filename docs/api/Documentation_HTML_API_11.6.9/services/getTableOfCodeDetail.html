Service générique : getTableOfCodeDetail


Ce service permet de donner le détail d'une nomenclature correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit.

  • getTableOfCodeDetail
    • TableOfCodeDetailRequest
      • requestHeader
        • requestId1
          • requestId : charMax50
        • serviceName : charMax50
        • timestamp : dateTime
        • originalName : charMax50
        • originalId : charMax50
        • originalTimestamp : dateTime
        • languageCode : languageCode
        • userCode : charMax10
        • channelCode : channelCode
      • getTableOfCodeDetailRequest : getTableOfCodeDetailRequest
        • tableOfCodeIdentifier : tableOfCodeIdentifier
          • tableNumber : char3
          • branchCode : char5
          • accessKey : charMax10
    • TableOfCodeDetailResponse
      • responseHeader
        • requestId1
          • requestId : charMax50
        • responseId : charMax50
        • timestamp : dateTime
        • serviceVersion : charMax10
        • language : language
          • code : char3
          • designation : charMax30
      • responseStatus
        • statusCode : statusCodeType
        • messages : responseStatusMessages
          • [M] message : responseStatusMessage
            • nature : responseMessageNature
            • code : char6
            • [M] line : string
            • errorInformation : errorInformation
              • errorCode : string
              • convertedCode : string
              • errorMessage : string
      • getTableOfCodeDetailResponse : getTableOfCodeDetailResponse
        • [M] tableOfCodeResponse : getTableOfCodeResponse
          • branch : branch
            • code : char5
            • designation : charMax30
          • tableNumber : tableOfCode
            • tableNumber : char3
            • designation : charMax35
          • accessKey : accessKey
          • [M] wordings : tableOfCodeWording
            • id : idWording
            • value : charMax30
          • [M] amounts : tableOfCodeAmount
            • id : idAmount
            • value : decimal15_0
          • [M] rates : tableOfCodeRate
            • id : idRate
            • value : decimal15_7

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
