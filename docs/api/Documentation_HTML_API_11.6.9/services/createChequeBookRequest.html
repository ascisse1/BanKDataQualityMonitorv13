Service générique : createChequeBookRequest

Tables amplitude :
  • bkadcli
  • bkcom

Ce service permet de créer une commande de chéquier correspondant aux critères de sélection contenus dans le flux XML qu'il reçoit si les contrôles métiers le permettent.

Attention, un service métier générique n’étant pas en mesure de communiquer avec SYNAPSE, le contrôle avec la Banque de France pour vérifier qu’un client n’est pas interdit bancaire est impossible. Il devra alors être géré dans l’application tierce appelant le service createChequeBookRequest.
