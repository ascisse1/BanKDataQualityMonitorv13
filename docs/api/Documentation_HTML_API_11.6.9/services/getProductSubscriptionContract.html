Programme Amplitude : cbwprdctrd
Service générique : getProductSubscriptionContract


Ce service permet de restituer toutes les données liées à la souscription d'un produit, pouvant être ensuite retravaillées par une application externe afin de produire une édition ou un autre document retranscrivant les informations liées à la vente.

Nom : getProductSubscriptionContractResponse
Description : Réponse de la restitution de la convention d'une vente de produit.
Commentaire :

Seules les éditions dont le code de la maquette, paramétrée dans les conventions de vente du produit, autorise le type du client ayant souscrit au produit sont générées.

Il est donc possible de n’avoir aucun flux à restituer pour une vente de produit, selon les maquettes paramétrées.
