Programme Amplitude : oiwswf940c
Service générique : createMT940Subscription


Ce service permet la création de souscription pour l'adressage des messages SWIFT MT940. Suite à la création, le numéro de souscription est renvoyé.

Le service utilise la fonction exposée de création d'une nomenclature.

  • createMT940Subscription
    • MT940SubscriptionRequest
      • requestHeader
        • requestId1
          • requestId : charMax50
        • serviceName : charMax50
        • timestamp : dateTime
        • originalName : charMax50
        • originalId : charMax50
        • originalTimestamp : dateTime
        • languageCode : languageCode
        • userCode : charMax10
        • channelCode : channelCode
      • createMT940SubscriptionRequest : createMT940SubscriptionRequest
        • branchCode : branchCode
        • designation : charMax30
        • customerReference : charMax16
        • bicCode : swiftBicCode
        • correspondentAccount : charMax18
        • generationFrequency : mt940GenFrequency
        • ribFormat : mt940RibFormat
        • interfaceCode : mt94xInterfaceCode
    • MT940SubscriptionResponse
      • responseHeader
        • requestId1
          • requestId : charMax50
        • responseId : charMax50
        • timestamp : dateTime
        • serviceVersion : charMax10
        • language : language
          • code : char3
          • designation : charMax30
      • responseStatus
        • statusCode : statusCodeType
        • messages : responseStatusMessages
          • [M] message : responseStatusMessage
            • nature : responseMessageNature
            • code : char6
            • [M] line : string
            • errorInformation : errorInformation
              • errorCode : string
              • convertedCode : string
              • errorMessage : string
      • createMT940SubscriptionResponse : createMT940SubscriptionResponse
        • mt940Identifier : accessKey

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
