  • otherCoHolderType
Nom : otherCoHolderType
Description : Autre type de co-titulaire.

Si cette balise est renseignée, alors le type de co-titulaire est forcé à Autre (A).

Choix : Non
  • otherCoHolderType
    • name : charMax36
    • firstName : charMax30
    • titleCode : char2

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
