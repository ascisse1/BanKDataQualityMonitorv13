Nom : dosprtCreateLoanFileRatesForConversion
Description : Taux utilisés pour les conversions.
Choix : Non
  • dosprtCreateLoanFileRatesForConversion
    • rateTypeUsedForConversion : dosprtRateType
    • rateTypeUsedForFundsRelease : dosprtRateType

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : rateTypeUsedForConversion
Description : Type de taux à appliquer lors des conversions dans une autre devise.
Commentaire :

Un type de taux de conversion par défaut doit être renseigné au niveau du type de prêt.

Le type de taux de conversion ne doit être renseigné que s’il peut être choisi (si le Montant 1 du paramètre CDTCONVERS en nomenclature 098 est positionné à 1), si on est  dans une banque croate (paramètre CROATIE en nomenclature 098) et si la réévaluation des dossiers est autorisée (paramètre CDTREEVAL en nomenclature 098).

Valeur par défaut : typetaux_conv

Nom : dosprtRateType
Description : Type de taux (utilisé pour les conversions).
Format : string
Valeurs possibles :
M00 Cours moyen
V00 Cours vente
A00 Cours achat
Nom : rateTypeUsedForFundsRelease
Description : Type de taux à appliquer lors des conversions au moment du déblocage des fonds.
Commentaire :

Le type de taux pour le déblocage ne doit être renseigné que s’il peut être  choisi (si le Montant 2 du paramètre CDTCONVERS en nomenclature 098 est positionné à 1), si on est  dans une banque croate (paramètre CROATIE en nomenclature 098) et si le type de déblocage au niveau du type de prêt est déblocage client (C), à la mise en place (M) ou immédiat (I).

Valeur par défaut : typetaux_debl

Nom : dosprtRateType
Description : Type de taux (utilisé pour les conversions).
Format : string
Valeurs possibles :
M00 Cours moyen
V00 Cours vente
A00 Cours achat
