Nom : thirdPartyPhoneNumber
Description : Téléphone d'un tiers.

Cette balise hérite des données de la requête "getThirdPartyPhoneNumberListRequest".

Choix : Non
  • thirdPartyPhoneNumber
    • thirdParty : restrictedThirdParty
      • thirdPartyNumber : thirdPartyCode
      • displayedName : charMax67
    • phoneType : phoneType
      • code : charMax3
      • designation : charMax30
    • phoneNumber : charMax20
    • formatOfTelephoneNumber : formatOfTelephoneNumber
      • code : charMax10
      • designation : charMax30

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
