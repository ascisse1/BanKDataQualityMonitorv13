  • modifyPEPDataRequest
Nom : modifyPEPDataRequest
Description : Requête pour la modification des informations PPE d'un client ou d'un tiers.
Choix : Non
  • modifyPEPDataRequest
    • [C] PEPCode : customerOrThirdPartyCode
      • customerCode : customerCode
      • thirdPartyCode : customerCode
    • PEPPosition : charMax3
    • PEPLinkLegacy : PEPLinkLegacy
      • linkCode : charMax3
      • legacyPEP : customerCode
    • mandateEndDate : date

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
