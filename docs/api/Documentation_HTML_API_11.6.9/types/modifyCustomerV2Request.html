Commentaire :

Si trouvées, ces informations seront tracées dans la table des historiques black liste.

Une opposition client est également créée, avec le code opposition paramétré dans le Montant 5 du paramètre BLACKLISTE en nomenclature 098 ou dans le Montant 5 du paramètre BLACKLNAT en nomenclature 098 si on ne contrôle qu’avec une black liste nationale (paramétré dans le Taux 5 égal à 1 du paramètre BLACKLISTE en nomenclature 098).

  • dirigeants correspondant au client dans la table des dirigeants,
  • actionnaires correspondant au client dans la table des actionnaires,
  • comptes joints du client dans la table des comptes joints,
  • groupes du client dans la table des groupes.
