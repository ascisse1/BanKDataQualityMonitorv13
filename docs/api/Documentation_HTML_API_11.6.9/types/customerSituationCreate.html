Nom : nationalityCode
Description : Code nationalité.
Commentaire :

Lors de la modification d’un client, si le code nationalité est modifié, alors il est mis à jour pour les :

  • dirigeants correspondant au client dans la table des dirigeants,
  • actionnaires correspondant au client dans la table des actionnaires.

Valeur par défaut : nat

Nom : applicationDateOfLegalSituation
Description : Date d'application de la situation juridique.
Commentaire :

La date d’application de la situation juridique ne peut être renseignée que si la situation juridique est renseignée.

Si elles sont renseignées, la date d'application de la situation juridique doit être postérieure à la date de naissance du client.

Si elles sont renseignées, la date d'application de la situation juridique doit être postérieure à la date de création de l’entreprise.

Elle doit être une date valide : l’année de la date d’application de la situation juridique ne doit pas être supérieure à l’année de la date comptable + 200.

Si la date d’application de la situation juridique est postérieure à la date comptable, les jours fériés de l’année correspondante doivent être paramétrés dans la table des jours fériés.

Valeur par défaut (contrôle de l’obligation de saisie uniquement) : dsitj
