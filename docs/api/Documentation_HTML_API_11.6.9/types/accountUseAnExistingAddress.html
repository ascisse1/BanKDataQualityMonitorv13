  • accountUseAnExistingAddress
Nom : accountUseAnExistingAddress
Description : Utilisation d'une adresse existante.
Choix : Oui
  • accountUseAnExistingAddress
    • existingAddressToBeUsed : accountExistingAddressToBeUsed
    • codingAddressType : charMax2

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : existingAddressToBeUsed
Description : Adresse existante à utiliser.
Nom : accountExistingAddressToBeUsed
Description : Adresse existante à utiliser.
Format : string
Valeurs possibles :
CC Adresse du compte
C Adresse courrier du client
D Adresse déclarative du client
Nom : codingAddressType
Description : Type d'adresse.
Commentaire :

Le type d'adresse doit correspondre à un type d’adresse client autre que ceux prévus dans le type "accountExistingAddressToBeUsed". 

Nom : charMax2
Description : Chaîne composée d'au plus 2 caractères.
Format : string
Longueur mini-maxi : 1 - 2
