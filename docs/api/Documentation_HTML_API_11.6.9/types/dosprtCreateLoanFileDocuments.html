Nom : dosprtCreateLoanFileDocuments
Description : Documents du dossier de crédit.
Choix : Non
  • dosprtCreateLoanFileDocuments
    • [M] loanFileDocument : dosprtCreateLoanFileDocument
      • documentCode : dosprtDocumentCode
      • documentIsMandatory : boolean
      • documentNatureCode : documentNatureCode
      • documentStepCode : documentStepCode

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : loanFileDocument
Description : Document.
Limite du nombre d'occurences mini-maxi : 1 aucune
Commentaire :

Des documents ne peuvent être renseignés que si le type de prêt le permet.

Nom : dosprtCreateLoanFileDocument
Description : Document.
Nom : documentCode
Description : Code du document.
Commentaire :

Le code document doit exister dans la table bkdocprt.

Contrôle sur l'unicité du code document en cas de plusieurs documents renseignés.

Nom : dosprtDocumentCode
Description : Code document.
Format : string
Longueur fixe : 3
Nom : documentIsMandatory
Description : Document obligatoire.
Commentaire :

Si ce champ n’est pas renseigné il est initialisé à FALSE.

Nom : boolean
Nom : documentNatureCode
Description : Nature du document.
Nom : documentNatureCode
Description : Code étape.
Format : string
Valeurs possibles :
1 Original
2 Copie
3 Copie certifiee
4 Duplicata
5 Manuscrit
Nom : documentStepCode
Description : Code de l'étape.
Commentaire :

La valeur 1 – "Etude du dossier" n'est pas autorisée pour la création/modification d'un dossier.

La valeur 2 – "Décision de la banque" n'est autorisée que si la comptabilisation de l'étape "Validation banque" est active.

La valeur 3 – "Décision du client" n'est autorisée que si la comptabilisation de l'étape "Validation client" est active.

Nom : documentStepCode
Description : Code étape.
Format : string
Valeurs possibles :
1 Etude du dossier
2 Decision de la banque
3 Decision du client
4 Validation du dossier
