  • documentaryRemittanceSituationType
Nom : documentaryRemittanceSituationType
Description : Situation de la REMDOC.
Format : string
Valeurs possibles :
O Ouverture
Tous
A Annulation
R Totalement réglé
E CREDOC export : Rappel de levée
F Clôture
L CREDOC export : Levée
M Modification

Liste des valeurs possibles :

  • A
  • L
  • F
  • M
  • R
  • E
  • O
  • " "

La signification de ce code dépend du type de REMDOC.

  • REMDOC export :
    • A = Annulation
    • F = Clôture
    • M = Modification
    • R = Totalement réglé
    • O = Ouverture
    • " " = Tous
  • REMDOC import :
    • A = Annulation
    • F = Clôture
    • M = Modification
    • R = Totalement réglé
    • O = Ouverture
    • " " = Tous
