Nom : mt940RibFormat
Description : Format RIB d'un MT940.
Format : string
Valeurs possibles :
0 Pas de formatage particulier : Numéro de compte Amplitude
1 Compte au format RIB
2 Compte au format IBAN

Liste des valeurs possibles :

  • 0 = Pas de formatage particulier, numéro de compte Amplitude
  • 1 = Compte au format RIB
  • 2 = Compte au format IBAN
