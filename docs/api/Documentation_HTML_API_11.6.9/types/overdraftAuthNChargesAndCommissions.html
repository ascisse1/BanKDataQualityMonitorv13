Légende :

Nom : chargesAmount
Description : Montant de frais.
Commentaire :

Le montant est en devise du compte. Si la balise est absente du flux d’entrée, les frais sont calculés à partir des conditions paramétrées pour les frais sur autorisation. Si des conditions de minimum ou maximum de frais sur autorisation sont définies, le montant de frais est modifié si nécessaire pour satisfaire ces contraintes.
