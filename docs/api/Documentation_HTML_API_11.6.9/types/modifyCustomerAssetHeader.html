  • modifyCustomerAssetHeader
Nom : modifyCustomerAssetHeader
Description : En-tête d'un patrimoine.
Choix : Non
  • modifyCustomerAssetHeader
    • customerCode : customerCode
    • referenceYear : short
    • reliabilityLevel : reliabilityLevel
    • exhaustiveInformation : boolean

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : customerCode
Description : Code client.
Commentaire :

Le code client n’est pas obligatoire si le service de modification du patrimoine d’un client est appelé par le service createCustomer, et est obligatoire dans tous les autres cas.

S’il est renseigné, le code client doit exister dans la table des clients.

Si le contrôle sur le contentieux est demandé dans le Montant 1 (égal à 1) du paramètre AWMH dans la nomenclature 098 ou si le paramètre AWMH n’existe pas dans la nomenclature 098, alors le client ne doit pas être présent dans la table des clients en contentieux.

Nom : customerCode
Description : Code client.
Format : string
Longueur mini-maxi : 1 - 15
Nom : referenceYear
Description : Année de référence du patrimoine.
Commentaire :

Valeur par défaut :

  •  patr_annee_p pour un client de type Particulier (1)
  • patr_annee_s pour un client de type Société (2) ou de type Entreprise individuelle (3)
Nom : short
Nom : reliabilityLevel
Description : Niveau de fiabilité du patrimoine.
Commentaire :

Valeur par défaut :

  • patr_fiabil_p pour un client de type Particulier (1)
  • patr_fiabil_s pour un client de type Société (2) ou de type Entreprise individuelle (3)
Nom : reliabilityLevel
Description : Niveau de fiabilité.
Format : string
Valeurs possibles :
F Fiable
M Moyen
N Non fiable
Nom : exhaustiveInformation
Description : Exhaustivité du patrimoine.
Commentaire :

Valeur par défaut :

  • patr_exhaus_p pour un client de type Particulier (1)
  • patr_exhaus_s pour un client de type Société (2) ou de type Entreprise individuelle (3)
Nom : boolean
