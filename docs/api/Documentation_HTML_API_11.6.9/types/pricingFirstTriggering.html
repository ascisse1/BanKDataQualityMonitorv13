Nom : pricingFirstTriggering
Description : Informations sur le premier déclenchement d'une tarification de produit ou package.
Choix : Non
  • pricingFirstTriggering
    • firstTriggeringType : pricingFirstTriggeringType
    • pricingTriggeringDay : short
    • proRata : boolean

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : firstTriggeringType
Description : Type de date du premier déclenchement de facturation périodique.
Nom : pricingFirstTriggeringType
Description : Type de date du premier déclenchement pour la tarification d'un produit ou package.
Format : string
Valeurs possibles :
0 Jour à saisir
1 Date de souscription
2 Date de souscription + 1 jour
3 Début du mois suivant la souscription
4 Fin du mois de souscription
5 Fin de période en cours
6 Début de mois, comptabilisation fin de mois précédent
Nom : pricingTriggeringDay
Description : Jour de tarification par défaut dans la période.
Nom : short
Nom : proRata
Description : Calcul du prorata pour le premier déclenchement.
Nom : boolean
