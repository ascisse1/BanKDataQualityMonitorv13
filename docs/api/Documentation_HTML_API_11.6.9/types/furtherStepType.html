  • furtherStepType
Nom : furtherStepType
Description : Complément d'étape.
Format : string
Valeurs possibles :
0 Pour étape étude : 0 = étude non validée / Pour étape validation : 0 = dossier validé et en attente de mise en place
1 Pour étape étude : 1 = étude validée / Pour étape décision banque : 1 = dossier accepté par la banque / Pour étape décision client : 1 = dossier accepté par le client / Pour étape validation : 1 = dossier mis en place
2 Pour étape étude : 2 = dossier abandonné lors de l'étude / Pour étape décision banque : 2 = dossier rejeté par la banque / Pour étape décision client : 2 = dossier rejeté par le client Pour étape validation : 0 = dossier validé et en attente de mise en place 1 = dossier mis en place 2 = dossier abandonné lors de la validation
3 Pour étape décision banque : 3 = refus de financement pour le dossier / Pour étape validation : 3 = Dossier en attente de compta clôture
4 Pour étape validation : 4 = Dossier en cours de déchéance
5 Pour étape validation : 5 = Dossier déchu
9 Pour étape validation : 9 = Dossier clôturé
