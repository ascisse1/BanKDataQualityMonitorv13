Nom : createLoanInstructionStepToProcessResponse
Description : Réponse d'une décision sur l'étape en cours d'un dossier IDC.
Choix : Non

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : decisionCompletionStatus
Description : Statut d'aboutissement de la décision.
Commentaire :

Valeurs possibles :

  • TRUE : la décision a abouti.
  • FALSE : la décision n'a pas pu aboutir.

En cas de décision non aboutie, le résultat des actions sera à consulter afin d'avoir le détail de la non-décision.
