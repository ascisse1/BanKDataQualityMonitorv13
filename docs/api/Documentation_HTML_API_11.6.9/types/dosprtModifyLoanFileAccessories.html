  • dosprtModifyLoanFileAccessories
Nom : dosprtModifyLoanFileAccessories
Description : Accessoires d¿un dossier de prêt.

Si un accessoire est renseigné, il doit être paramétré au niveau des accessoires du type de prêt.

Par exemple, si le poste 3 est renseigné représentant la commission 1, alors la troisième ligne des accessoires doit être paramétrée dans la rubrique "Intérêts et accessoires" du type de prêt.

Choix : Non

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Nom : capital
Description : Capital.
Nom : dosprtModifyDetailCapital
Description : Capital.
Nom : capitalAmount
Description : Montant de capital.
Commentaire :

Le montant du capital doit être compris entre la valeur minimum et la valeur maximum des conditions CAPttt ou CAPMAX, où ttt correspond au code du type de prêt.

Si le dossier est à comptabiliser (comptabilité étude, comptabilité banque, comptabilité client ou comptabilité à la validation finale) et si le montant du prêt a changé, alors un événement CRSOUR est ajouté à la table des événements.

Nom : decimal19_4
Description : Type décimal à 19 chiffres dont 4 décimales.
Format : decimal
Nombre total de chiffres : 19
Nom : capitalMaturity
Description : Terme du capital.
Commentaire :

Valeur par défaut : ter_cap

Nom : dosprtDetailMaturityType
Description : Terme d'un poste.
Format : string
Valeurs possibles :
A Perçu d'avance
E A l'échéance
Nom : capitalPeriodicity
Description : Périodicité du capital.
Commentaire :

Valeur par défaut : per_cap

La périodicité du capital, si elle est renseignée, doit être un multiple du nombre de mois du prêt (durée du prêt calculée par différence entre la date de dernière et la date de première échéance, ou durée du prêt renseignée directement par le nombre d'échéances).

Nom : dosprtDetailPeriodicityType

Format : integer
Min-max inclusive : 1 - 12

Nom : dosprtCreateDetailCommission
Description : Commission.
Nom : commissionAmountOrRate
Description : Montant ou taux de la commission.
Commentaire :

La balise renseignée (Taux ou Montant) doit être en accord avec le paramétrage (M ou T) de la commission  dans les conditions COxttt ou COxPRT où ttt correspond au code du type de prêt, et x correspond à 1, 2 ou 3 selon la commission 1, 2 ou 3.

Le montant ou le taux de la commission x doit être compris entre la valeur minimum de la condition MICxPR et la valeur maximum de la condition MACxPR où x correspond à 1, 2 ou 3 selon la commission 1, 2 ou 3.
