  • legalAttribute
Nom : legalAttribute
Description : Attribut légal.

L'attribut légal doit exister dans la table de paramétrage des attributs légaux pour le type de dossier.

Choix : Non
  • legalAttribute
    • identifier : char3
    • value : charMax10

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : identifier
Description : Identifiant de l'attribut.
Nom : char3
Description : Type avec un nombre de caractères égal à 3.
Format : string
Longueur fixe : 3
Nom : value
Description : Valeur de l'attribut.
Commentaire :

La valeur de l'attribut n’est modifiable que si le paramétrage l’autorise.

La valeur de l'attribut doit exister dans la nomenclature dont le numéro de table correspond à l'identifiant de l’attribut.

Nom : charMax10
Description : Chaîne composée d'au plus 10 caractères.
Format : string
Longueur mini-maxi : 1 - 10
