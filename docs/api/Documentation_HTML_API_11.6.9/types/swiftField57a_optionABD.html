Nom : swiftField57a_optionABD
Description : Ce type définit le champ 57a d'un message SWIFT avec option A, B ou D.
Choix : Non
  • swiftField57a_optionABD
    • [C] swiftField57a_optionABD : swiftParticipantIdentifierWithCodeAddressOrLocation

        • partyIdentifier : swiftPartyIdentifier

        • nameAndAddress : swiftNameAndAddress
          • [M] line : charMax35
      • identifierWithLocation : swiftparticipantIdentifierWithLocation
        • partyIdentifier : swiftPartyIdentifier
        • location : charMax35

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.
