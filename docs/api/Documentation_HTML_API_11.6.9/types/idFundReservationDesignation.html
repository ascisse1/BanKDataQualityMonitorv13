  • idFundReservationDesignation
Nom : idFundReservationDesignation
Description : Numéro de libellé de réservation de fonds.
Format : short
Min-max inclusive : 1 - 3

Liste des valeurs possibles :

  • 1 = Premier libellé (première zone de saisie libre) de la réservation de fonds.
  • 2 = Deuxième libellé (deuxième zone de saisie libre) de la réservation de fonds.
  • 3 = Troisième libellé (troisième zone de saisie libre) de la réservation de fonds.
