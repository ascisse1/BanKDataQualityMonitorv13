Nom : addressLine3
Description : Ligne 3 de l'adresse.
Commentaire :

Cette balise ne doit être fournie que si le format de l’adresse est GE ou CX.

Si le format de l’adresse est GE, si le paramètre NCALEDONIE en nomenclature 098 existe et si le paramètre FMT-ADRGEO existe en nomenclature 098 avec le Montant 3 renseigné et différent de 0 :

  • Si le paramètre NCALADRGEO en nomenclature 098 est renseigné avec le Montant 3 à 1 pour les particuliers ou le Montant 5 à 1 pour les sociétés ou pour les entreprises individuelles le Montant 7 à 1 ou le Montant 1 à 1, alors le numéro de la rue correspondant aux 4 premières positions de la ligne 3 de l’adresse doit être renseigné.
  • Sinon, le numéro de rue correspondant aux 4 premières positions de la ligne 3 de l’adresse doit être numérique.
  • Si le paramétrage NCALADRGEO en nomenclature 098 est renseigné avec le Montant 4 à 1 pour les particuliers ou le Montant 6 à 1 pour les sociétés ou pour les entreprises individuelles le Montant 8 à 1 ou le Montant 2 à 1, alors le nom de la rue contenu à partir de la 6ème position de la ligne 3 de l’adresse doit être renseigné.

Sinon, le nom de la rue contenu à partir de la 6ème position de la ligne 3 de l’adresse doit être alphanumérique.

Nom : charMax30
Description : Chaîne composée d'au plus 30 caractères.
Format : string
Longueur mini-maxi : 1 - 30
Nom : city
Description : Ville.
Commentaire :

Cette balise ne doit être fournie que si le format de l’adresse est BP, SP, GE ou CX.

Si le format de l’adresse est GE et si le paramètre FMT-ADRGEO existe en nomenclature 098 avec le Montant 5 renseigné et différent de 0,  ou si le format de l’adresse est BP ou SP :

  • Si le paramètre NCALEDONIE existe en nomenclature 098, alors la ville doit être renseignée  et alphanumérique.
  • Valeur par défaut :
    • villeb si le format de l’adresse est BP.
    • villes si le format de l’adresse est SP.
    • villeg si le format de l’adresse est GE.
  • Si le Montant 3 est à 1 pour le paramètre VILLE en nomenclature 098, alors la ville doit exister dans la table des villes.

Nom : postalCode
Description : Code postal.
Commentaire :

Cette balise ne doit être fournie que si le format de l’adresse est BP, SP, GE ou CX.

Si le format de l’adresse est BP ou SP, et si le Libellé 2 est à O pour le paramètre VILLE en nomenclature 098, alors le code postal doit être alphanumérique.

Si le format de l’adresse est CX :  

  • Valeur par défaut : cposcx.
  • Le code postal doit exister en nomenclature 080 et la ville doit être égale au Libellé 2.

Si le format de l’adresse est GE, si le paramètre NCALEDONIE existe en nomenclature 098 et si le paramètre FMT-ADRGEO existe en nomenclature 098 avec le Montant 4 renseigné et différent de 0 :

  • Le code postal doit être alphanumérique.
  • Valeur par défaut : cposg.
