Nom : Authorisation1Choice
Description : Type de code lié à une autorisation.
Choix : Oui
  • Authorisation1Choice
    • Cd : Authorisation1Code
    • Prtry : Max128Text

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : Authorisation1Code
Description : Code autorisation.
Format : string

ILEV Instruction concernant le niveau d'autorisation
AUTH Fichier pré-autorisé
FDET Fichier indiquant le détail des niveaux d'autorisation
