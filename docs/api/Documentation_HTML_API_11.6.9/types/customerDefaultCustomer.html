  • customerDefaultCustomer
Nom : customerDefaultCustomer
Description : Défaut client.

La gestion du défaut n’est autorisée que si le paramètre B2-DEFCLI est présent en nomenclature 098.

La gestion du marquage du défaut n’est possible que si les segments de clientèle sont paramétrés en nomenclature 708.

La saisie du défaut client n’est possible que si le niveau de forçage de l’utilisateur est supérieur au niveau de forçage pour la saisie du défaut, paramétré dans le Montant 1, ou au niveau de forçage pour la modification du défaut client, paramétré dans le Montant 2 du paramètre B2-DEFCLI en nomenclature 098.

Le choix des 2 balises retail/non retail doit être en accord avec le type de défaut du client.

Le code segment est paramétré dans la valeur alphanumérique de la donnée complémentaire du client pour laquelle :

  • l’attribut est égal au type de segmentation paramétré dans le Libellé 3 de la nomenclature 171 pour la valeur de l’information complémentaire du segment contenue dans le Libellé 5 du paramètre B2-DEFCLI en nomenclature 098
  • le nom est :
    • soit égal à SEG si l’attribut de la donnée complémentaire est spécifique (S),
    • soit paramétré dans le Libellé 5 du paramètre B2-DEFCLI en nomenclature 098, dans les autres cas.

Le type de défaut du client est retail si le Montant 3 (sur 2 caractères) du code segment en nomenclature 708 est égal à la valeur de l’identifiant retail paramétrée dans le Libellé 4 du paramètre B2-DEFCLI en nomenclature 098.

Le code de la cotation est alimenté avec le code qualité du client.

Choix : Oui
  • customerDefaultCustomer
    • retail : customerRetail
      • pmlDefault : boolean
      • pmlDefaultEntry : date
      • nonPmlDefault : boolean
      • nonPmlDefaultEntry : date
    • nonRetail : customerNonRetail
      • default : boolean
      • defaultEntry : date

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
