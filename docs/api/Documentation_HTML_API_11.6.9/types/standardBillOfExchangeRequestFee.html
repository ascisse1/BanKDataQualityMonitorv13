Nom : standardBillOfExchangeRequestFee
Description : Caractéristiques des frais sur la demande de lettre de change.
Choix : Non
  • standardBillOfExchangeRequestFee
    • amount : decimal19_4
    • valueDate : date

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : amount
Description : Montant des frais sur la demande de lettre de change.
Commentaire :

Cette valeur est définie par la condition FRDLxx (FRDLCN par défaut) ; il est possible de définir une valeur et celle-ci sera prise si elle est comprise dans les variations de la condition.

Nom : decimal19_4
Description : Type décimal à 19 chiffres dont 4 décimales.
Format : decimal
Nombre total de chiffres : 19
Nom : valueDate
Description : Date de valeur de la prise de frais.
Commentaire :

Cette valeur est définie par la condition DATVAL ; il est possible de définir une valeur et celle-ci sera prise si elle est comprise dans les variations de la condition.

Nom : date
