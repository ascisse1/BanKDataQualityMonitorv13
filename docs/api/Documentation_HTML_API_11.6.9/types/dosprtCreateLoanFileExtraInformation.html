Nom : dateValue
Description : Valeur information de type date.
Commentaire :

Si cette donnée est renseignée, alors :

  • Le type de donnée, indiqué dans le Libellé 5 de la nomenclature 167, doit être D.
  • Si les Montant 7 et Montant 8 de la nomenclature 167 sont tous les deux valorisés, alors la date doit :
    • Etre antérieure à la date comptable à laquelle est ajoutée la valeur du Montant 7 en nombre d’années.
    • Etre postérieure à la date comptable à laquelle est retirée la valeur du Montant 8 en nombre d’années.
