Nom : nextCardStatus
Description : Prochain statut d'une carte.
Format : string
Valeurs possibles :
A Carte annulée
FJ Fabrication rejetée
E Carte échue
F Carte en fabrication
G Carte remise au client
H Carte plastifiée hors agence
I Carte plastifiée en agence
R Carte réceptionnée en agence
BP Carte bloquée par le prestataire
FV Fabrication validée
N Aucun renouvellement ni aucune réfection en cours
Z Carte en réfection
X Demande de carte envoyée en production
U Renouvellement manuel
V Demande de carte validée
