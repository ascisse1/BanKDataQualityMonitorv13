Nom : accountproduct
Description : Compte produit.
Choix : Non
  • accountproduct
    • productCode : productCode
    • packageCode : packageCode

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : productCode
Description : Code produit.
Commentaire :

Le code produit doit exister dans la table des produits.

Seuls les comptes rattachés à des produits d'attribut "compte" ou les comptes rattachés à des dossiers sont autorisés dans le cadre de ce service.

S'il s'agit d'une création de compte rattaché à un produit d'attribut "compte", les contrôles d'éligibilité du client à la souscription du produit sont effectués (contrôles identiques à ceux réalisés dans le cadre d'une souscription de produit).

Nom : productCode
Description : Code produit.
Format : string
Longueur fixe : 3
Nom : packageCode
Description : Code package.
Commentaire :

Cette information concerne les comptes rattachés à des produits d'attribut "compte" uniquement.

Si le code package est renseigné, les contrôles d'éligibilité du client à la souscription du produit/package sont effectués (contrôles identiques à ceux réalisés dans le cadre d'une souscription de produit).

Nom : packageCode
Description : Code package.
Format : string
Longueur fixe : 6
