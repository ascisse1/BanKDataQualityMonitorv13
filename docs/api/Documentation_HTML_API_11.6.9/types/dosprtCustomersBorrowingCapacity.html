Nom : dosprtCustomersBorrowingCapacity
Description : Quotité cessible disponible.

La quotité cessible disponible ne doit être renseignée que si le client est une personne physique (si son code intitulé existe en nomenclature 036 avec un Montant 1 égal à Personne physique (1) ou Entreprise individuelle (3)).

Choix : Non
