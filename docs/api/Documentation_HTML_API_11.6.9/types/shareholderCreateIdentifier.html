  • shareholderCreateIdentifier
Nom : shareholderCreateIdentifier
Description : Identifiant d'un actionnaire d'un client.
Choix : Non
  • shareholderCreateIdentifier
    • customerCode : customerCode
    • shareholderCode : shareholderCode
    • shareholderType : customerOrThirdPartyType

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : customerCode
Description : Code du client.
Commentaire :

Le code client n’est pas obligatoire si le service de création d’un actionnaire d’un client est appelé par le service createCustomer, et est obligatoire dans tous les autres cas.

S’il est renseigné, le code client doit correspondre à un client existant dans la table des clients.

Si le contrôle sur le contentieux est demandé dans le Montant 1 (égal à 1) du paramètre AWMH dans la nomenclature 098 ou si le paramètre AWMH n’existe pas dans la nomenclature 098, alors le client ne doit pas être présent dans la table des clients en contentieux.

Le client ne doit pas être de type Particulier (1).

Le couple formé par le code actionnaire et le type d’actionnaire doit être unique pour le client.

Nom : customerCode
Description : Code client.
Format : string
Longueur mini-maxi : 1 - 15
Nom : shareholderCode
Description : Code actionnaire.
Commentaire :

Si le type d’actionnaire désigne un client (C), alors le code actionnaire doit identifier un enregistrement dans la table des clients.

Sinon, si le type d’actionnaire désigne un tiers (T), alors le code actionnaire doit correspondre à un enregistrement dans la table des tiers.

Valeur par défaut du nom de l’actionnaire : noma

Valeur par défaut du prénom de l’actionnaire : prea

Valeur par défaut de la nationalité de l’actionnaire : nata

Si le contrôle de l’actionnaire est demandé dans le paramètre CONTROLDIR en nomenclature 098 avec le Montant 2 renseigné à 1, alors au moins un actionnaire doit être renseigné pour le client.

Le contrôle d’obligation de saisie d'au moins un actionnaire est aussi paramétrable via l’une des valeurs par défaut suivantes :

  • intia
  • prea
  • nata
  • pcap
  • marche
