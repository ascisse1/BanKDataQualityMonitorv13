Nom : fileaccountNature
Description : Nature de compte dossier.

Élément permettant de déterminer le chapitre comptable d'un compte dossier à partir des chapitres paramétrés via la gestion des chapitres par nature de compte cbgeschnc.

Rappel : en fonction de ce paramétrage, le service ne procèdera pas systématiquement à la création d'un compte mais pourra ramener un compte existant.

Choix : Non
  • fileaccountNature
    • moduleCode : char3
    • accountNature : char3
    • fileType : char3

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
