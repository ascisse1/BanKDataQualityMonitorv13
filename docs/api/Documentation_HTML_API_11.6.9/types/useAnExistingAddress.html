  • useAnExistingAddress
Nom : useAnExistingAddress
Description : Utilisation d'une adresse existante.
Choix : Oui
  • useAnExistingAddress
    • existingAddressToBeUsed : existingAddressToBeUsed
    • codingAddressType : charMax2

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : existingAddressToBeUsed
Description : Choix de l'adresse existante parmi des types fixes.
Nom : existingAddressToBeUsed
Description : Choix de l'adresse parmi des types fixes.
Format : string
Valeurs possibles :
CC Adresse compte
C Adresse courrier client
S Adresse spécifique
D Adresse déclarative client
CH Adresse chéquier
Nom : codingAddressType
Description : Choix de l'adresse existante parmi des types paramétrés en nomenclature 180.
Commentaire :

Le type d'adresse doit exister en nomenclature 180.
Le client titulaire du compte doit posséder une adresse de ce type.
 

Nom : charMax2
Description : Chaîne composée d'au plus 2 caractères.
Format : string
Longueur mini-maxi : 1 - 2
