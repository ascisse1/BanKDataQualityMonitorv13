Nom : cardFreeOperations
Description : Gratuité des opérations.
Choix : Non
  • cardFreeOperations
    • transactionNature : operationNatureCode
    • frequency : cardFreeOperationsFrequency
    • type : gratuityType
    • value : decimal19_4
    • finalEndDate : date

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : transactionNature
Description : Nature de l'opération.
Commentaire :

Doit appartenir à la liste des opérations (module étape) pour le type de la carte.

Nom : operationNatureCode
Description : Code nature opération.
Format : string
Longueur fixe : 6
Nom : frequency
Description : Périodicité de la gratuité.
Nom : cardFreeOperationsFrequency
Description : Fréquence de la gratuité des opérations monétiques
Format : string
Valeurs possibles :
A Annuelle
B Bi-Hebdomadaire
S Semestrielle
T Trimestrielle
M Mensuelle
H Hebdomadaire
J Journalière
Nom : type
Description : Type de gratuité.
Nom : gratuityType
Description : Type de gratuité.
Format : string
Valeurs possibles :
1 Nombre d'opérations
2 Montant maximum de l'opération
3 Cumul du montant des opérations
Nom : value
Description : Nombre/montant.
Nom : decimal19_4
Description : Type décimal à 19 chiffres dont 4 décimales.
Format : decimal
Nombre total de chiffres : 19
Nom : finalEndDate
Description : Date de fin définitive.
Commentaire :

Doit être postérieure ou égale à la date comptable.

Nom : date
