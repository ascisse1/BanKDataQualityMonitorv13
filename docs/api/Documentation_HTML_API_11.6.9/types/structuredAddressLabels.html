Nom : structuredAddressLabels
Description : Balises d'un format d'adresse structurée.
Choix : Non
  • structuredAddressLabels
    • [M] addressLabel : structuredAddressLabel
      • labelName : charMax250
      • mandatory : boolean
      • tolerance : decimal3_0

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : addressLabel
Description : Information de chaque balise d'un format adresse structurée.
Limite du nombre d'occurences mini-maxi : aucune aucune
Nom : structuredAddressLabel
Description : Information de chaque balise d'un format adresse structurée.
Nom : labelName
Description : Libellé de la balise pour le format - utilisation du libellé par défaut de la balise si ce n'est pas renseigné.
Nom : charMax250
Description : Chaîne composée d'au plus 250 caractères.
Format : string
Longueur mini-maxi : 1 - 250
Nom : mandatory
Description : Indication de donnée à renseigner obligatoirement lors de la création d'une adresse structuré.
Nom : boolean
Nom : tolerance
Description : Pourcentage de tolérance d'erreur de la balise pour le format.
Nom : decimal3_0
Description : Type décimal à 3 chiffres dont 0 décimales.
Format : decimal
Nombre total de chiffres : 3
