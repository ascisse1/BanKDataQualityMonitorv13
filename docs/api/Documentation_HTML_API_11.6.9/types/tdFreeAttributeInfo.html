  • tdFreeAttributeInfo
Nom : tdFreeAttributeInfo
Description : Informations sur un attribut libre DAT.

Les attributs libres sont définis partiellement dans la nomenclature 167.

Choix : Non
  • tdFreeAttributeInfo
    • code : charMax10
    • designation : charMax30
    • operation : operation
      • code : char3
      • designation : charMax30
    • tdMandatoryInput : boolean
    • dataFormat : freeAttributeFormat
    • totalLengthIfAlpha : integer
    • numberOfDecimalsIfAmountOrRate : short

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
