Nom : rootCardMiniStatementsFiles
Description : Fichiers de mini relevés d'une carte.
Choix : Non
  • rootCardMiniStatementsFiles
    • [M] miniStatementsFile : cardMiniStatementsFile
      • bankIdentifier : institutionCode
      • [M] miniStatement : cardMiniStatement
        • accountIdentifier : internalAccountKey
          • branch : char5
          • currency : currencyCode
          • account : charMax11
          • suffix : charMax2
        • checkKey : charMax2
        • transactionSequenceNumber : integer
        • amount : decimal19_4
        • transactionSide : side
        • operationLabel : charMax255
        • operationCode : operationCode
        • transactionDate : date
        • transactionValueDate : date
        • eventNumber : eventNumber
        • transactionNature : operationNatureCode
        • transactionModuleCode : moduleCode

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
