  • letterOfGuaranteeProcessingCode
Nom : letterOfGuaranteeProcessingCode
Description : Code traitement d'une caution
Format : string
Valeurs possibles :
0 Caution en attente de forçage (AT) ou post-datée (VA)
2 Caution mise en place
4 Caution échue (dech) - non levée totalement
6 Caution mise en jeu - la banque paie à la place du client
8 Caution en suspens, puis une fois comptabilisée passe à 7
1 Caution validée ou forcée. Compta non effectuée
3 Caution en cours
5 Caution en cours - sans date d'échéance
7 Caution en suspens - comptabilisée
9 Caution échue - levée totalement
