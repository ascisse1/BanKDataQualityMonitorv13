  • dosprtCustomerFundsReleasePaymentMethod
Nom : dosprtCustomerFundsReleasePaymentMethod
Description : Méthode de déblocage des fonds pour un mode de déblocage client.
Format : string
Valeurs possibles :
A Autre compte
R Remboursement
C Chèque
E Espèces
V Virement
W Swift MT202
S Compte de restructuration

La méthode de déblocage des fonds sur le compte de remboursement du client n’est modifiable que si le remboursement des échéances du dossier de crédit se fait sur le compte de remboursement du client.
