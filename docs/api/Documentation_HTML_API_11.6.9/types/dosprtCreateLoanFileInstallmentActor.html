Nom : customerOrThirdPartyCode
Description : Code client ou tiers concerné.
Commentaire :

Si l’intervenant est un tiers et qu’un client a été créé à partir de ce tiers, alors l’intervenant est considéré comme un client qui doit exister dans la table des clients.

La longueur du code client ou du code tiers doit correspondre aux longueurs  paramétrées respectivement dans le Montant 1 du paramètre CLIENT et dans le  Montant 1 du paramètre CREA-TIERS en nomenclature 999.

Si l’intervenant est un client, si l’ajout ou la suppression de co-emprunteurs ou de cautions ne peuvent pas se faire sans contrôle du profil (si le  Montant 5 du paramètre CDTPAR2 en nomenclature 098 est différent de 1) alors   un contrôle d’habilitation de l’utilisateur par rapport au profil de clientèle et un contrôle d’autorisation du chapitre sont effectués.

Chaque code client ou tiers ne doit être présent qu’une seule fois parmis la liste des co¿emprunteurs ou cautions renseignés.

Nom : paymentMethod
Description : Mode de règlement des échéances.
Commentaire :

Le mode de règlement des échéances doit exister en nomenclature 124.

Si le Libellé 4 de la nomenclature 124 est à P et que l’on n’est pas en mode hors banque (paramètre CDTSANSBQE absent en nomenclature 098), alors le Libellé 3 doit être différent de O.

Le déblocage sur travaux ou sur facture doit être différent de A ou F et la commission d’engagement doit être différente de C, M, T ou P.

Nom : distributedRate
Description : Taux de répartition.
Commentaire :

Le taux de répartition est obligatoire et sa valeur doit être comprise entre 0 et 100.

Le taux de répartition permet de calculer :

  • Soit Le montant d’engagement pour l’intervenant pour un plan d’amortissement constant,
  • Soit le taux d’engagement pour l’intervenant pour un plan d’amortissement non constant.

Nom : borrowingLimit
Description : Capacité d'emprunt du client.
Commentaire :

La capacité d’emprunt du client est prise en compte si le calcul de la quotité cessible disponible est demandé pour le type de prêt.

Pour un intervenant de type caution (C), le montant de la quotité cessible disponible calculé ou de la capacité de financement doit être supérieur ou égal au montant de l’échéance constante du prêt si le contrôle bloquant de la QCD est demandé (Montant 2 du paramètre PRET en nomenclature 098 différent de 0).

Pour un intervenant de type co-emprunteur (E), le total des montants de la quotité cessible disponible calculé ou de la capacité de financement doit être supérieur ou égal au montant de l’échéance constante du prêt si le contrôle bloquant de la QCD est demandé (Montant 2 du paramètre PRET en nomenclature 098 différent de 0).
