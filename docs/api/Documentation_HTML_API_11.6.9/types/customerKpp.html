Nom : customerKpp
Description : KPP.

La gestion des informations KPP (Banque RUSSE) n'est autorisée que si le paramètre RUSSIE est présent en nomenclature 098.

Le nombre maximum d'informations KPP autorisées pour un client est de 99.

Choix : Non
  • customerKpp

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Description : Numéro de KPP.
Limite du nombre d'occurences mini-maxi : aucune aucune
Commentaire :

Le numéro de KPP doit contenir 9 caractères et être unique pour le client.
