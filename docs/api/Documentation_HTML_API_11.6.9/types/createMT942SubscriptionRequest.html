Nom : correspondentAccount
Description : Compte du correspondant.
Commentaire :

Le compte saisi est de la forme : code agence sur 5 caractères suivi du numéro de compte et du suffixe éventuel. Le compte doit exister. Le couple code BIC et compte du correspondant doit être unique.

Nom : creditAnyAmount34F
Description : Montant "Crédit/Tous sens" du 34F.
Commentaire :

Indique le montant minimum au "Crédit" ou "Tous sens" du champ 34F pour générer le message SWIFT MT942.

Nom : decimal15_0
Description : Type décimal à 15 chiffres dont 0 décimales.
Format : decimal
Nombre total de chiffres : 15
Nom : debitAmount34F
Description : Montant au "Débit" du 34F.
Commentaire :

Indique le montant minimum au "Débit" du champ 34F pour générer le message SWIFT MT942. La saisie de cette information n'est autorisée que si la balise "creditAnyAmount34F" est renseignée.

Nom : decimal15_0
Description : Type décimal à 15 chiffres dont 0 décimales.
Format : decimal
Nombre total de chiffres : 15
Nom : ribAccountFormat
Description : Compte au format RIB.
Commentaire :

Indique si le compte doit être formaté comme un RIB dans le champ 25 des messages SWIFT MT942.
