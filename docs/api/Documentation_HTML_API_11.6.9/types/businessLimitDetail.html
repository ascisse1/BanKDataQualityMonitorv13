Nom : defaultAmount
Description : Montant par défaut du plafond.
Commentaire :

Ce montant doit être exprimé dans la devise du compte principal de la carte.

Le montant par défaut du plafond doit être supérieur à 0.

Dans le cas d’un plafond compte, si la donnée complémentaire positionnée dans le Montant 8 du paramètre MON_PLASIS dans la nomenclature 098 indique que le plafond n’est pas modifiable, alors la valeur saisie sera ignorée.

Nom : decimal19_4
Description : Type décimal à 19 chiffres dont 4 décimales.
Format : decimal
Nombre total de chiffres : 19
Nom : exceptionalAmount
Description : Montant exceptionnel du plafond.
Commentaire :

Ce montant doit être exprimé dans la devise du compte principal de la carte.

Le montant exceptionnel du plafond doit être supérieur à 0.

Dans le cas d’un plafond compte, si la donnée complémentaire positionnée dans le Montant 8 du paramètre MON_PLASIS dans la nomenclature 098 indique que le plafond n’est pas modifiable, alors la valeur saisie sera ignorée.

Si un plafond exceptionnel existe déjà pour le compte ou la carte et que cette balise n’est pas renseignée, alors le plafond exceptionnel sera supprimé.

Nom : decimal19_4
Description : Type décimal à 19 chiffres dont 4 décimales.
Format : decimal
Nombre total de chiffres : 19
Nom : startDate
Description : Date de début du plafond exceptionnel.
Commentaire :

Si elle est renseignée, la date de début du plafond exceptionnel doit être inférieure ou égale à la date de fin du plafond exceptionnel, si cette dernière est également renseignée.

Nom : date
Nom : endDate
Description : Date de fin du plafond exceptionnel.
Commentaire :

Si elle est renseignée, la date de fin du plafond exceptionnel doit être supérieure ou égale à la date comptable, et supérieure ou égale à la date de début du plafond exceptionnel, si cette dernière est également renseignée.

Nom : date
