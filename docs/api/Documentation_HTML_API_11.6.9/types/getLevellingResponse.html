Nom : accountConsideration
Description : Compte de contrepartie.
Commentaire :

Dans le cas d’un transfert de compte à compte au débit multi bénéficiaires, la saisie de la totalité des informations constituant l’identifiant du compte est nécessaire.

  • ¿ Pour un compte interne à la banque et au format interne : valoriser l’agence, la devise, le numéro de compte et l’éventuel suffixe du compte.
  • ¿ Pour un compte issu d’une autre banque nationale et au format interne : valoriser le code établissement, le code guichet et le numéro de compte.
  • ¿ Pour un compte issu d’une banque étrangère : valoriser, en plus de l’IBAN ou du format libre, soit le BIC de la banque, soit le code établissement et le code guichet.
