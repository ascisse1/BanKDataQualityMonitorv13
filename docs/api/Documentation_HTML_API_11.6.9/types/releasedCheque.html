Nom : releasedCheque
Description : Informations sur un chèque débloqué.
Choix : Non
  • releasedCheque
    • chequeSerialNumber : char2
    • chequeNumber : charMax8
    • chequeAmount : decimal19_4

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Nom : chequeSerialNumber
Description : Numéro de série du chèque.
Commentaire :

Le numéro de série du chèque est obligatoire si le Libellé 2 du paramètre CHEQUES dans la nomenclature 098 est égal à O.

Nom : char2
Description : Type avec un nombre de caractères égal à 2.

Longueur fixe : 2
Nom : chequeNumber
Description : Numéro de chèque.
Commentaire :

La longueur du numéro du chèque doit être égale au Montant 3 du paramètre CHEQUES dans la nomenclature 098.
Le chèque :

  • ¿ Ne doit pas faire l'objet d’un autre évènement.

  • ¿ Doit être en opposition.
  • ¿ Doit être en stock.

Nom : chequeAmount
Description : Montant du chèque.
Commentaire :

S'il est renseigné, le montant doit être égal au montant du chèque à débloquer.

Nom : decimal19_4
