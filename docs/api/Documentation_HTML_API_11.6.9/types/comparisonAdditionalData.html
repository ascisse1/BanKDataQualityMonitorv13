Nom : comparisonAdditionalData
Description : Structure de base d'une donnée complémentaire.
Choix : Non
  • comparisonAdditionalData
    • identifier : charMax10
    • [C] value : comparisonAdditionalDataValue
      • alphanumeric : designation
        • value : string
        • comparisonOperator : comparisonOperator
      • amountOrRate : comparisonDecimal
        • decimal1 : decimal32_16
        • decimal2 : decimal32_16
        • comparator : comparator
      • date : comparisonDate
        • date1 : date
        • date2 : date
        • comparator : comparator

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques

Description : 2ème décimale.
Commentaire :

Le 2ème décimal n’est pris en compte que pour les opérateurs de comparaison BETWEEN et BETWEEN EQUALS.

N'est jamais renseigné dans un flux de réponse d’un service.
