Description : Type de compte.
Format : string
Valeurs possibles :
SLRY Compte utilisé pour le paiement de salaire
CISH Compte utilisé pour le paiement des revenus si différent du compte courant (espèces)
SVGS Compte d'épargne
CACC Compte courant
ODFT Compte utilisé pour les autorisations de découvert
COMM Compte de commission (si différent du compte du transfert)
NREX Compte utilisé pour les non résidents extérieurs
MGLD Compte utilisé pour des locations marginales
ONDP Compte utilisé pour les dépôts à vue
LOAN Compte utilisé dans un dossier de prêt
CASH Compte utilisé pour des paiements en espèces
TRAS Compte utilisé pour le commerce si différent du compte courant
CHAR Compte de frais (si différent du compte du transfert)
SACC Compte de règlement
MOMA Compte utilisé pour les marchés financiers
TAXE Compte utilisé pour les taxes si différent du compte du transfert
