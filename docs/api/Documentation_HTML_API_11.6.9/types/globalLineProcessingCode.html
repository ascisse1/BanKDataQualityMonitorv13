  • globalLineProcessingCode
Nom : globalLineProcessingCode
Description : Ce type définit le type de recherche de limites effectué (avec ou sans contrôle, avec ou sans calcul de l'encours).
Format : string
Valeurs possibles :
U calcul des encours sans contrôle ou
R lecture sans calcul ni contrôle d'encours
C calcul et le contrôle des encours

Liste de valeurs : R, U, C

R pour la lecture sans calcul ni contrôle d'encours

U pour le calcul des encours sans contrôle ou

C pour le calcul et le contrôle des encours
