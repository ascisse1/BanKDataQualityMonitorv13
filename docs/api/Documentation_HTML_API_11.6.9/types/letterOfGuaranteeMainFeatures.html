  • letterOfGuaranteeMainFeatures
Nom : letterOfGuaranteeMainFeatures
Description : Ce type contient les caractéristiques générales d'un dossier de caution.
Choix : Non
  • letterOfGuaranteeMainFeatures
    • subject : charMax30
    • dates : letterOfGuaranteeDates
      • dateOfIssue : date
      • calculationStartDate : date
      • maturityDate : date
    • frequencies : letterOfGuaranteeFrequencies
      • frequencyOfCommission : letterOfGuaranteeFrequency
      • nextFrequencyOfCommission : letterOfGuaranteeNextFrequency
      • frequencyOfDeductions : letterOfGuaranteeFrequencyOfDeductions
    • collateralCreditInLGCurrency : boolean
    • realDays : boolean
    • proRataTemporisCommission : boolean
    • automaticRelease : boolean
    • pledgeOfCash : boolean

Légende :

  • = Elément obligatoire en saisie.
  • [M] = Elément multiple (0 à n occurrences).
  • [C] = Elément composé d'un choix à effectuer. Un seul choix autorisé.

Remarque : Un même élément peut regrouper plusieurs caractéristiques
