Service SOAP : getDocumentaryCreditList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/lettersofcredit/v1/
Description : The service getDocumentaryCreditList allows returning a list of letters of credit (L/C) matching the search criteria received. The search is based on the optional criteria below: - the L/C identifier: branch (code and/or designation), module, L/C type (code and/or designation), file number, - the internal reference, - the L/C availability mode (code and/or designation), - the currency: alphanumeric code and/or digital code and/or designation, - the file account: number (internal and/or IBAN and/or external format) and/or class and/or customer and/or additional information (identifier and/or value (or value range) and/or designation), - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the customer name, - the situation, - the opening date (of a date range), - the validity date (or a date range), - the amount (or an amount range), - the L/C status, - the transferable credit flag, - the revocable credit flag, - the standby flag, - the confirmed credit flag, - the issuing mode, - the interface code, - the L/C reference, - the irregularity flag, - the payment flag, - the last bank comment flag. For each L/C returned, all data used in the search criteria will be indicated.
