Insurance conditions that can be applied to a loan file
  • loaninsuranceconditions

Service SOAP : getAmortizableLoanInsuranceConditionsDetail
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/loaninsuranceconditions/v1/
Description : The service getAmortizableLoanInsuranceConditionsDetail allows returning detailed information about insurance conditions that can be applied to a loan file corresponding to a single identifier received. Information sent back is linked to insurance conditions about commissions: - Insurance premium - Insurer premium, - Extra premium.
