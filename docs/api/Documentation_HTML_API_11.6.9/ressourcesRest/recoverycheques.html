Cheques for recovery
  • recoverycheques
    • v1
      • GETLIST
Service SOAP : getRecoveryChequesList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/recoverycheques/v1/
Description : The service getRecoveryChequesList allows returning a list of cheques for recovery matching the search criteria received. The search is based on the optional criteria below: - information on the cheque remittance: branch (code and/or designation), reference, remittance date (or date range), remitting customer (code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference), remitter's account (number (internal or IBAN or external format) and/or class and/or additional information (identifier and/or value (or value range) and/or designation)), credit value date (or date range), remittance slip number, - information on the cheque issuer: bank institution (code and/or designation), branch (code and/or designation), account (number (internal and/or IBAN and/or external format) and/or class and/or customer (code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference) and/or additional information (identifier and/or value (or value range) and/or designation), or other bank account), - information on the cheque for recovery: serial number, cheque number (or number range), currency (code and/or designation), amount (or amount range), - information on the remittance processing: event number, origin, operation (code and/or designation), operation nature (code and/or designation), recovery branch (code and/or designation), presentation date (or date range), collection method, cheque fate (code and/or date (or date range)), rejection reason (code and/or designation), - information on the cheque collection: Direct Credit outstanding account (number (internal or IBAN or external format) and/or class and/or customer (code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference) and/or additional information (identifier and/or value (or value range) and/or designation), unavailability date or date range), collection account (number (internal and/or IBAN and/or external format) and/or class and/or customer (code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference) and/or additional information (identifier and/or value (or value range) and/or designation). For each cheque returned, all data used in the search criteria will be indicated.
