Service SOAP : getCollateralList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/collaterals/v1/
Description : The service getCollateralList allows returning a list of collateral files matching the search criteria received. The search is based on the optional criteria below: - the collateral identifier: branch (code and/or designation), collateral number, - the reference, - the collateral nature (code and/or designation), - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the commitment: type (code and/or designation), branch (code and/or designation), commitment number, - the guarantor: guarantor code, covered percentage type, type (customer/third party), code, name to be displayed, - the currency (alphanumeric code and/or digital code and/or designation), - the creation date (or a date range), - the expiry date (or a date range), - the collateral status. For each collateral file returned, all data used in the search will be indicated, except for the commitment and guarantor information, as well as: - the collateral amount, - the allocated amount, - the last allocation date.
