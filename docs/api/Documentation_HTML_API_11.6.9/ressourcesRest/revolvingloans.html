Service SOAP : modifyRevolvingLoan
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/revolvingloans/v1/{loanId}
Description : The service modifyRevolvingLoan allows modifying the revolving credit file corresponding to a unique identifier received. The data that can be modified depends on the credit reimbursement type: - either the reimbursement amount, - or the reimbursement percentage (in this case, the service recalculates and updates the reimbursement amount).

Service SOAP : getRevolvingLoanList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/revolvingloans/v1/
Description : The service getRevolvingLoanList allows returning a list of revolving credit files matching the search criteria received. The search is based on the optional criteria below: - the revolving credit identifier: branch (code and/or designation), file number, amendment number, - the revolving credit type (code and/or designation), - the step and/or step complement, - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer(code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the activation, - the ceiling amount (or an amount range), - the implementation date (or a date range), - the first instalment date (or a date range). For each revolving credit file returned, all data used in the search criteria will be indicated as well as: - the drawing end date, - the borrower type, - the available amount, - the interest rate and the APR.
