Service SOAP : getThirdPartyDetail
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/thirdparties/v1/{thirdPartieId}
Description : The service getThirdPartyDetail allows getting detailed information on the third party holder corresponding o a unique identifier received. For the selected third party, the following information is returned: - details of the third party, - details of addresses of the third party, - the list of telephone numbers of the third party, - the list of email addresses of the third party, - details of the free attributes of the third party.

Service SOAP : getThirdPartyList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/thirdparties/v1/
Description : The service getThirdPartyList allows returning a list of third parties matching the search criteria received. The search is based on the optional criteria below: - the third party: code and/or name to be displayed, type, national identifier, additional information (identifier and/or value (or value range) and/or designation), external reference, - the third party type, - details of an individual third party: last name, first name, middle name, birth date (or date range), identity record (type and/or number), - details of a company third party: corporate name, logo, trade registration number, business licence number, tax identity number. For each third party returned, all data used in the search criteria are indicated.
