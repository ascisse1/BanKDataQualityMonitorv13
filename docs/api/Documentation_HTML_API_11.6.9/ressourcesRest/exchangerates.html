  • exchangerates

      • GETLIST
Service SOAP : getExchangeRateList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/exchangerates/v1/
Description : The service getExchangeRateList allows returning a list of exchange rates matching the search criteria received. The search is based on the optional criteria below: - the branch code, - the initial currency code and/or target currency code, - the operation code, - the customer profile code, - the rate application date (by default: the ledger date), - the origin of the rate, - the search nature. For each rate returned, the following information will be indicated: - the calculation method, - the rate application date, - the exchange rate.
