Description : The verb GET applied to URI /loaninstructiontypes /v1/{loaninstructiontypeId}/authorisedcodifieddata return the list of the authorized codified data for a loan instruction type. Among this detail, defined in the setup through a list of values, we find : financing currencies, file items currencies, goods and costs natures, resources natures, goods for sale natures, additional date natures, sureties natures, financial data natures, financing natures and Portal tasks.

Uri : http://soprabanking.com/amplitude/loaninstructiontypes/v1/{loaninstructiontypeId}/externalsteps
Description : The verb GET applied to URI /loaninstructiontypes /v1/{loaninstructiontypeId}/externalsteps return the list of steps to be processed by an external procedure for a loan instruction type.
