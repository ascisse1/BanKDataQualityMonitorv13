Service SOAP : getStoppedChequesList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/chequestoppages/v1/
Description : The service getStoppedChequesList allows returning a list of stopped cheques matching the search criteria received. The search is based on the optional criteria below: - the branch (code and/or designation), - the serial number, - the cheque number (or a number range), - the holder's account: number (internal and/or IBAN and/or external format) and/or class and/or customer (code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference) and/or additional information (identifier and/or value (or value range) and/or designation), - the stoppage setting reason (code and/or designation), - the stoppage application date (or a date range), - the stoppage release date (or a date range), - the stoppage input date (or a date range). For each cheque returned, all data used in the search criteria will be indicated as well as: - the stoppage release reason (code and designation), - the cheque amount, - the beneficiary name, - the cheque issue date.
