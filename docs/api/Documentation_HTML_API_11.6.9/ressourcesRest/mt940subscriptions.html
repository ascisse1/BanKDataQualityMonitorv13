Service SOAP : cancelMT940Subscription
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/mt940subscriptions/v1/{MT940Id}
Description : The service cancelMT940Subscription allows cancelling a subscription to the routing of MT940 messages corresponding to a unique identifier received (update of table of codes 401 of MT940 and MT950 messages). If some information concerning the MT950 messages is defined in the record corresponding to the identifier received, then only the information related to the MT940 messages are deleted; the identifier and the information related to MT950 messages are kept.
