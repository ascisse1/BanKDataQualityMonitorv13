Service SOAP : getDirectDebitAuthNList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/directdebitauthorizations/v1/
Description : The service getDirectDebitAuthNList allows returning a list of direct debit authorisations matching the search criteria received. The search is based on the optional criteria below: - the paying organisation: code, designation, account number, national issuer number, - the subscriber number, - the account: number (internal and/or IBAN and/or external format) and/or class and/or customer (populationFile) and/or additional information (identifier and/or value (or value range) and/or designation), - the additional identifier, - the branch (code and/or designation). For each direct debit authorisation returned, all data used in the search criteria will be indicated as well as: - the creation date, - the reason, - the limit (amount and currency of expression), - the termination date, - the suspension start date, - the suspension end date.
