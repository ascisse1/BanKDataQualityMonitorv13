Service SOAP : getChequeBookRequestList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/chequebookrequests/v1/{chequeBookRequestId}
Description : The service getChequeBookRequestList allows returning a list of cheque book requests matching the search criteria received. The search is based on the optional criteria below: - the branch (code and/or designation), - the request number, - the account number (internal and/or IBAN and/or external format), - the cheque book dispatch mode, - the request date (or a date range), - the customisation date (or a date range), - the delivery date (or a date range), - the destruction date (or a date range), - the last modification date (or a date range), - the request situation, - the request status, - the cheque book delivery branch (code and/or designation), - the total number of cheque books ordered by branch and by request number. For each cheque book request returned, all data used in the search criteria will be indicated.

Service SOAP : createChequeBookRequest
Code retour succes : 201 - Created
Uri : http://soprabanking.com/amplitude/chequebookrequests/v1/
Description : The service createChequeBookRequest allows creating a cheque book order. The order characteristics are required: branch, account, origin, dispatch mode, dispatch place, one or several types of cheque book (with for each type : number of cheque books, attributes and title). The service returns: - the order number, - the order date, - the order value date, - the order situation, - the order status, - the fees amount, - the disapproval list (code and designation).
