Service SOAP : getTableOfCodeListV2
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/tablesofcodes/v2/
Description : The service getTableOfCodeList allows returning a list of keys of tables of codes matching the search criteria received. The search is based on the optional criteria below: - the branch (code and/or designation), - the table of codes (number and/or title), - the access key, - a table of wordings of tables of codes (identifier and/or value), - a table of amounts of tables of codes (identifier and/or value), - a table of rates of tables of codes (identifier and/or value), For each line of table of codes returned, all data used in the search criteria will be indicated.
