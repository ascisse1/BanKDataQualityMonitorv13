Service SOAP : getCurrencyList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/currencies/v1/
Description : The service getCurrencyList allows returning a list of currencies matching the search criteria received. The search is based on the optional criteria below: - the alphanumeric currency code, - the digital currency code, - the currency designation. For each currency returned, all data used in the search criteria will be indicated as well as: - the country (code and designation), - the number of decimals, - the cleared currency indicator, - the number of decimals for rate rounding, - the currency filter indicator, - the in/out currency, - the local code, - the number of debit interest days, - the number of credit interest days, - the 'yesterday' price variation percentage, - the fixed parity rate, - the consolidation coefficient, - the Euro conversion rate, - the CRM display currency indicator, - the exchange rate.
