Service SOAP : cancelStandingOrderDomesticTransfer
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/standingorderdomtransfers/v1/{transferId}
Description : The cancelStandingDomesticTransferValidation API allows you to cancel a permanent transfer. The API must receive the identifier of the original request in order to recognize the original transfer. It is also necessary to indicate the reason for the cancellation of the permanent transfer. The API will return the status of the transfer and indicate whether the transfer has been created or not, and the reason for rejection encountered when the transfer wasn¿t created. Here is the list of the different statuses: ACTC : Transfer correctly integrated but not processed ACSP : Integrated and properly processed transfer PDNG : Transfer processed but with an error (The error encountered will be described in the response) RJCT : Transfer rejected

Service SOAP : createStandingOrderDomesticTransfer
Code retour succes : 201 - Created
Uri : http://soprabanking.com/amplitude/standingorderdomtransfers/v1/
Description : The createStandingOrderDomesticTransfer API allows you to create a simple permanent transfer between two accounts with an amount and a reason for the transfer. The transfer may be for internal use (same agency or other agency) or external use (other bank). However, this transfer must remain within the local perimeter, that is to say to a local bank. The API will return the status of the transfer and indicate whether the transfer has been created or not, and the reason for rejection encountered when the transfer wasn¿t created. Here is the list of the different statuses: ACTC : Transfer correctly integrated but not processed ACSP : Integrated and properly processed transfer PDNG : Transfer processed but with an error (The error encountered will be described in the response) RJCT : Transfer rejected
