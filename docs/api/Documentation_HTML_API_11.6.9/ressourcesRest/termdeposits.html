Service SOAP : getTDDetail
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/termdeposits/v1/{termDepositId}
Description : The service getTDDetail allows getting detailed information on a TD corresponding to a unique identifier received. For the selected TD, the following information is returned: - details of the TD, - the list of amendments (detailed information), - the list of generations of amendments (detailed information), - the list of increasing interest rates, - the list of free attributes.

Service SOAP : getTDList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/termdeposits/v1/
Description : The service getTDList allows returning a list of term deposits matching the search criteria received. The search is based on the optional criteria below: - the identifier: branch (code and/or designation), TD number, - the currency (alphanumeric code and/or digital code and/or designation), - the TD type (code and/or designation), - the account to be debited: number (internal and/or IBAN and/or external format) and/or class and/or customer and/or additional information (identifier and/or value (or value range) and/or designation), - the customer to be debited: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the pledging indicator, - the value date (or a date range), - the maturity date (or a date range), - the processing code. For each term deposit returned, all data used in the search criteria will be indicated as well as: - the deposit account (internal, IBAN and external formats), - the execution date, - the initial amount, - the rate, - the capitalisation indicator.
