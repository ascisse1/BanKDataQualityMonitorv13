Service SOAP : getUserPermissionCheck
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/userpermissionchecks/v1/
Description : The getUserPermissionCheck service allows you to control user permissions on a customer and / or an account. For a customer, the check is performed on the manager and profile. For an account, the check is performed on the type and class of the account.
