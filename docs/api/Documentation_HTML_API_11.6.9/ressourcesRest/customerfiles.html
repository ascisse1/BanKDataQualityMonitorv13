Customer global image files
  • customerfiles
    • v1
      • GETLIST
Service SOAP : getCustomerFileList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/customerfiles/v1/
Description : The service getCustomerFileList allows returning a list of global image files matching the search criteria received. The search is based on the optional criteria below: - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or nom), additional information (identifier and/or value (or value range) and/or designation), external reference, - the module (code and/or designation), - the branch maintaining the account (code and/or designation), - the file identifier, - the file type, - the file currency (alphanumeric code and/or digital code and/or designation), - the implementation date (or a date range), - the expiry date (or a date range), - the expression currency of the result (alphanumeric code and/or digital code and/or designation). For each global image file returned, all data used in the search criteria will be indicated as well as: - the original amount of the file, - the outstanding amount of the file, - the original amount in the currency of expression, - the outstanding amount in the currency of expression.
