Service SOAP : getDocumentaryRemittanceList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/importdocumentcollections/v1/
Description : The service getDocumentaryRemittanceList allows returning a list of documentary collection files (D/C) matching the search criteria received. The search is based on the optional criteria below: - the D/C identifier: branch (code and/or designation), module, D/C type (code and/or designation), file number, - the internal reference, - the interface, - the currency: alphanumeric code and/or digital code and/or designation, - the file account: number (internal and/or IBAN and/or external format) and/or class and/or customer and/or additional information (identifier and/or value (or value range) and/or designation), - the customer: code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference, - the customer name, - the situation, - the opening date (or a date range), - the validity date (or a date range), - the amount (or an amount range), - the D/C status. For each D/C returned, all data used in the search criteria will be indicated.
