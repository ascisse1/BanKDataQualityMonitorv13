Account sweeping files and account-to-account transfer files
  • levellings
    • v1
      • GETLIST
Service SOAP : getLevellingList
Code retour succes : 200 - OK
Uri : http://soprabanking.com/amplitude/levellings/v1/
Description : The service getLevellingList allows returning a list of account sweeping files and account-to-account transfer files matching the search criteria received. The search is based on the optional criteria below: - the account sweeping identifier: branch (code and/or designation), operation (code and/or designation), event number, - the account sweeping type (code and/or designation), - the triggering account: number (internal and/or IBAN and/or external format) and/or class and/or customer (code and/or name to be displayed, type, profile (code and/or designation), national identifier, account officer (code and/or name), additional information (identifier and/or value (or value range) and/or designation), external reference) and/or additional information (identifier and/or value (or value range) and/or designation), - the contra account (internal or other bank or international account), - the application date (or a date range), - the due date (or a date range), - the account sweeping type indicator. For each account sweeping or funds transfer returned, all data used in the search criteria will be indicated as well as: - the triggering type, - the rules for triggering a transfer with one debit and several credits: indicator of all credits, type of account sweeping amount, - the triggering threshold, - the last transfer date, - the priority, - the frequency, - the multi-beneficiary indicator.
